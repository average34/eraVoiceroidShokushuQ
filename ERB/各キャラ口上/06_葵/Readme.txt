この口上は東方触手宮リグルちゃん口上をほんのり葵ちゃん風に改変した物です
大本のリグルちゃん口上が男の娘パッチに対応していたため、意図せず葵君にも対応してしまいました
私は男の娘パッチONにしないから検証出来てないのでバグったらごめんね、言ってくれれば直します
いくつかオリジナル要素として、茜ちゃんにだけ個別対応口上を作ってあります
茜ちゃんには丁寧語で喋らないで欲しい！　って言う想いから追加しましたが、全部は無理です
下記改変箇所に追加した所を明記しておくので他の部分も追加したい人は頑張って下さい
私もやる気が出たら頑張ります
CASE 5を他の対応数字（2ならマキマキ、8ならきりたん）に変更したらその子専用の口上に書き換えられます
CASE 5からCASEELSEの上までをコピペして、CASE 5を書き換えたら茜ちゃん専用口上残したまま増やせます
ゆかあお、あおマキ、あおきり、あおウナ、好きなカップリングを増やしてください

東方触手宮リグルちゃん口上
----------------------------------------------------------------------

・諸注意

・残酷表現がHard以上でリグルちゃんのHPが1/5以下のとき、敵から通常攻撃を受けた時の口上が変化します。
いわゆるリョナ口上ですが、死亡・切断・流血などの表現はほぼありませんし、R-18レベルの残酷な表現はありません（と思ってます）
ガイドラインを参照して自己判断したので、問題があればお手数ですが連絡をください

・戦闘中のほとんどの口上は発情or淫乱orその部位の中毒スキルLv3or触手中毒Lv3orおねだり時で分岐します。
手っ取り早く分岐口上を見たいときは媚薬を持たせるといいと思います

・この口上は未完成です。
埋めようと思ってたけどまだ埋めてない口上　→　MAP表示時、触手服、出会い頭治療（責め側）、OP＆ED、ドッペル全身攻撃など細かいところと新しいモンスターの口上など
モチベ次第ではまた書くかもしれませんが……代わりに埋めてくれる人も募集してます

・催眠に関する設定は口上本体の最初の方に書いてあります。そちらを参照してください
