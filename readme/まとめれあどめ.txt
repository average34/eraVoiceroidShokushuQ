=================================================
	東方触手宮用まとめパッチ
=================================================
東方触手宮用に作られたパッチをまとめた上で調整などを行ったものです
既に何かパッチをあててる状態のバリアントに適用しても正しく動作しないかもしれません
また、正しく適用したとしても正常に動作するとは限りませんのであしからず


『おねだり』『ふたなり』『触手スーツ』『お酒』『蒸着』の5つのパッチを統合してみました。
いつの間にか、『ふたなり勝手に追加』『使い魔』も統合されてた件。
8/3　『抱き着き』『娘他種族化』統合。新パッチ『発情レベルパッチ』を追加。
8/18	酒の人が『消費武器二刀流』『胸が育たなくなる』パッチを追加。
8/25	『アクセサリ使用』『あなたの素質変更』『霊撃』パッチも追加。


大半がお酒パッチ作者様からの流用になってます。もう足向けて寝れねぇぜ。
