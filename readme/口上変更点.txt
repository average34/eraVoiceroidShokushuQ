●全体の仕様変更について

同キャラの複数の口上を同梱できるように、KOJO_キャラ番号_通し番号 という形式で管理することになりました。
サンプルの半霊口上辺りを見れば複数キャラがどう管理されるのか分かると思います。

また、口上セレクタで選択する時のために、
口上のタイトルや説明が設定できるようになっています。
（これもサンプルの半霊などを見れば分かるかと）

同梱口上はReadmeなどからこちらで適当なタイトル・説明をつけていますが、
何か不都合がありましたらご連絡ください。

今後は、本体に同梱する際に私の方で通し番号を付けて行こうと思います。
(勿論最初から他の口上と被らないように番号を付けておいてくれても構いません)


通し番号は0がデフォルトとなり、9まで使えます。
(流石に10種類は超えないだろうということでこうしてあります)

基本的に最初に公開された口上から順に番号を振っていきますのでご了承ください。

なお、サンプルの上海・半霊口上は数に数えませんので、
これらのキャラの口上を作る際は、普通に0から使って貰ってOKです。



一応補足（サンプル口上などを見て貰えれば分かると思いますが）

タイトルを設定したい場合
@KOJO_TITLE_キャラ番号_通し番号
を作成し、RESULTSにタイトルを入れてください

解説を設定したい場合
@KOJO_DETAIL_キャラ番号_通し番号
を作成し、表示したい内容をPRINTFORMLなどで表示してください
