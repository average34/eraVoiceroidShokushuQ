東方触手宮のライセンスについて


まず、ぱにめーしょん部分と各キャラの口上については
触手宮作者には権利が無いとし、それぞれの作者のReadme等に従うようにしてください。

本体に取り込まれている各種パッチについては、
「触手宮作者がその内容を全て確認・把握したうえで、必要ならば調整を加えて統合している」
ため、色々とややこしくならないよう、触手宮本体の一部として扱わせてください。
（勝手な事するな！というパッチ作者さんが居られましたら、スレなどでご指摘ください）


で、これらの内容を踏まえて、ぱにめーしょんと口上を除いた
「触手宮本体」部分については、大体以下のような感じで扱って貰えばOKです。

●別人による修正・加筆・改変
　OKです。むしろより良い作品にしていくためにも、推奨です。
　拡張の方針などについても一切制限しません。
　各人が好きなものを作って、好きなものを導入して遊んでください。
　
　長文を読むのが苦にならない人は、「東方触手宮誕生秘話.txt」を読んでみると
　触手宮作者が何を考えて作っていたのかが記載されているので、参考くらいにはなるかもしれません。

●別人による二次配布
　OKです。

●触手宮をベースとしたバリアント等の開発
　OKです。

●触手宮を元ネタとした二次創作
　やる人が居るのかどうか分かりませんが、eratohoJ で話題になった事がったので一応。
　本体側としては有償無償を問わず二次創作全般を許可します。
　ただし、先述のように口上部分に関しては各口上作者に権利がありますので、
　それらの内容を利用する際は、各口上作者に許可をもらってください。



なお、ここで言うのも変な感じですが、私の過去の作品である
eratohoL, eratohoJ, eratohoP, D&T
についても同様のものとして扱って貰ってOKです。
（J+ や D&Tex は別の方の作品なので、当然対象外です）

特にP以前のものはロダが死んでいて既にDLできなくなっていますので、
必要があれば二次配布等、自由にやっちゃってください。
（私自身は自分用に手を加えたものしか手元に残っていないので、「最新版」は持っていなかったりします）
