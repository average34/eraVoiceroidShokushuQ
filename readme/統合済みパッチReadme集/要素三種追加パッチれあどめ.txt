触手宮1012用要素三種追加パッチ （製作者　喚く狂人）Ver1.0　初版

・概要
	「初期ボーナスペット」「追加ボーナス」「服装効果」という三つの要素を追加します。易化を目的としており、総合的に見て難易度は低下します。
	また、まとめパッチの「現在未修正のバグ一覧」にあったバグを一つ修正します（たぶん）。
	１．初期ボーナスとしてペットを追加
		初期ボーナスとして「ペットをもらう」が追加されます。番号70「小悪魔」がもらえます。
		この小悪魔は初期能力がキャラ並に高く、かつ満腹度と好感度が最高から始まるので頼れる序盤の味方になってくれるはずです。飯食わせとけば。
	２．初期ボーナスとは別に、キャラ固有の追加ボーナスを追加
		低能力キャラでの序盤の辛さを改善するための変更です。極序盤の運ゲーさを緩和する追加ボーナスを設けました。内容はキャラによって固定です。
		選出基準は大まかに、「戦闘素質が40以下であり、戦闘素質+速度が100を割っていること」ですが、早苗などの例外もあります。
	３．服装効果を追加
		衣服によって色々な補正を得られるようになりました。メリットとデメリットが存在し、高いメリットには高いデメリットがついて回ります。
		補正は計算時にのみ用いられ、ステータス表示に出てくることはないです。
		なおこの変更に伴い、衣服選択時の地の文が多少変更されています。
	４．バグ修正
		まとめパッチ記述の「ペットが居ないのに、迷宮内で出会ったキャラをペットに襲わせることができるバグ」を修正しました（たぶん）。
		拘束具つけた相手と会話する時、SIFが書かれてなかったのが原因でした。
