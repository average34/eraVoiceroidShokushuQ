触手宮101用汚染度拡張パッチ （製作者　喚く狂人）Ver1.0　初版

・概要
	汚染度の仕様を拡張します。
	汚染度が戦闘行動に及ぼす影響の強化、戦闘中の汚染度の自然上昇、汚染行動の威力上昇、汚染度に応じた敵HPの変化等の変更を加えます。
	これにより、汚染度を念頭に入れて戦う必要がでてきます。高い汚染度では難化、低い汚染度では易化するバランスです。
	また、戦闘中に汚染度と浄化力を表示するようにし、浄化が行えるようにしています。ただしこの浄化は効率が悪く、低確率で暴走します。
	汚染度上昇の機会が増えているので、救済策として浄化用アイテム「浄化玉」を追加しました。所持限界は３です。
