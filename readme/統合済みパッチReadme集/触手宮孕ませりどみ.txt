今、はやりの東方触手宮。拘束されていると中出し攻撃されるわけですが……
分類が寄生攻撃。運が悪いと寄生される。寄生。寄生……？
いや違う。なんか違う。個人的に違う。やっぱり孕ませでしょっっっっっっ！！！　ｺﾞﾒﾝﾅｻｲ

というわけで寄生と妊娠を別個の存在として、中出しされると低確率で孕むようになるパッチ。
ついでに寄生攻撃を別に作ったり。
導入方法はv0.03に上書き。

触手宮が公開されて間もないためあまり複雑に組むとそちらの開発を阻害すること、
及び自分の実力がないという理由でめっちゃ簡素な出来。
CFLAG:60（射精量）とCFLAG:36（妊娠フラグ）、CFLAG:37（妊娠残り時間）を新たに使用。

・射精を受けた量
　CFLAG:60で管理。戦闘後にリセット。敵の技で射精するものがあればそれぞれのERBに追記する感じ。
　だいたい（現在存在する中出しが普通の射精だとして）中出し攻撃を一回受けることに３増える。
　ただし小型体型持ちは増加量が１少ない。また、触手中毒があるとそのレベル分一発ごとに増える。
　素股とかが実装されたら１～２、大量射精とか来たら４～５でいいんじゃないかな？
　バイオプラントを実装する際には４～５射精をバシバシ搭載するとよさそう。
　ちなみに中出し攻撃は性攻撃扱いにしている。

　……やっぱいちいち技ごとに書くよりTFLAG:55に中出し属性追加してTFLAG:76に技ごとの射精量追加して、
　FIX_BATTLE_ACTIONでCFLAG:60にTFLAG:76を追加したほうがいいのかしらん？
　ただその場合、威力計算とかどう調整すればいいかちょっとわからんのでできなかった。

・妊娠確率
　AFTER_BATTLEで管理。処理がめんどくさいので終了時拘束されている場合は妊娠しないことにする。
　基本は(CFLAG:60)/30。薬毒耐性か蓬莱人があると確率半減。寄生されていると妊娠しない。
　母乳体質だと確率倍加、触手中毒があれば更にレベル＋１倍加。
　普通だとあまり妊娠しない（中出し攻撃自体受ける場合が少ないし）が、犯されまくっていると危ない。
　戦闘後すぐは妊娠が発覚しないしステ欄にも表示されないが、こっそりCFLAG:36とCFLAG:37が立つ。
　個人的には、寄生が植えつけた側の影響で子供が変化するなら妊娠は植えつけられた側で変化……
　と分けたいので何に妊娠させられたかは記録しない。

・妊娠期間
　基本的に、寄生より期間は長い。治療方法は産む以外になく、妊娠中は寄生を受けない。
　一定期間経つと妊娠していることが明らかになる（その前にも怪しい文章が出るが……）。
　その際、CFLAG:36は１から２に変動。ステ欄にも表示されるようになる。
　また、出産後ではなく妊娠中に母乳体質を習得する。あとは寄生とだいたい同じ。
　後述の理由により現在ペナが弱いので、長期間寄生よけになることを考えるともっとペナ欲しいかも？
　妥当なラインとしては妊娠発覚時点でもMPダメージとか、妊娠が進むごとに速度ダウンとかだろうか。
　一応、CFLAG:36の変化は現在３まで用意してあるので補正追加＆変化は容易だと思う。

・寄生攻撃
　従来の中出し（800）を孕ませ攻撃にして、寄生攻撃である801を作成。性能は同じ。出る条件も頻度も同じ。
　ただテスト触手は寄生も孕ませもやるので捕まった時の危険性は高くなった。

・今後の課題
　個人的には誰が孕んでも生まれる候補としてドッペルゲンガー入れたいなーとか考えてるので、
　当該モンスターがいない現在はモンスター追加処理を削っている。
　企画書によればドッペルゲンガーはかなり強いようなので、妊娠するような場合は少なく設定した。
　ドッペルゲンガー以外の候補としては新規に「○○の娘」な触手使いモンスターを作るというのもありだが、
　名前だけ変えてデータは同一化するとしても重労働が目に見えている……
　代替としては「ゆうかりんが孕んだらアルラウネ系も候補に入る」とか、キャラごとに系統作るくらいか。
