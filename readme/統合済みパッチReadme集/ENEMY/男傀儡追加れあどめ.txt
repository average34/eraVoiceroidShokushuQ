触手宮101用男傀儡追加パッチ （製作者　喚く狂人）Ver1.1　初版

・概要
	敵として男傀儡（おとこくぐつ）というキャラを追加します。ランクは4、種類は32（その他）です。
	「触手宮改変で男に輪姦されるバリアント出ないかな」という声のをたまに聞く気がしたので作ったものです。バリアントではないですが。
	湧かせる場合はコンフィグからフラグを立ててください。

・敵の説明
	触手宮にうっかり迷い込んでしまった男が触手により自我を乗っ取られ、少女をいたぶるための道具に作り替えられたのが男傀儡です。
	遠目に見れば人間の男に見えますが、近くで見ると、異様に膨れ上がった筋肉や、そそり立ち我慢汁を垂れ流すやはり異様なサイズのペニスなどで、明らかな様子のおかしさが分かることでしょう。
	男傀儡は性質としては触手そのものであり、射精するための肉穴（＝少女たち）を探して常に迷宮をうろついている、まさにこの迷宮の傀儡です。
	一度肉穴を捉えると高確率で押し倒そうとするか、そうでなかったとしても何かしらの性的な行動に出ようとします。押し倒されたが最後、触手で強化された膂力で押さえつけられ、傀儡の射精欲が収まるまで「使われて」しまうことでしょう。
	また、男傀儡は邪魔をされることを非常に嫌い、肉穴で射精するときに邪魔されると、怪力で邪魔者を積極的に排除しようとしてきます。この時は相手が肉穴だろうと関係なく、暴力を振るいます。
	頑丈で力強く処女を奪われやすく、しかもランク的にちょくちょく出てくる面倒な相手です。しかし幸いにも、触手が肉体をそれほど上手く操作できていないため、足が遅いという欠点があります。
