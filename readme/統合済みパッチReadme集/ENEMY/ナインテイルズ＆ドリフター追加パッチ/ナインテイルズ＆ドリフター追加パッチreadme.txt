東方触手宮Ver1.00.2＋まとめパッチ用新モンスター追加パッチ

・本パッチは東方触手宮Ver1.00.2＋まとめパッチに新モンスター「ナインテイルズ」＆「ドリフター」を追加するパッチです。
　概要は以下のQ&Aを御覧ください。

・Q&A

Q.ナインテイルズってどんなモンスター？

A.子供でありながら高い能力を持つ、九尾の狐です。
　しかし9本の尻尾のうち8本は触手が擬態したものであり、コウノトリと同様に触手と狐が共生する事で、本物の九尾の狐並みの力を身に付けています。
　ただ、力を身に付けていても子供は子供。性格は無邪気で人懐っこく、まだまだお母さんのおっぱいを飲みたいお年ごろなのです。
　ちなみにメスです。　
　尻尾を使った攻撃を仕掛けてきたり狐火で服を燃やしてきたりと、化け狐らしい攻撃を揃えていますが、非常に厄介なのは人に化ける能力です。
　少女に変化して自慰を見せつける事でパーティ全体を発情させたり、大男に変化してアックスボンバーを放ったりと、かなりフリーダムな攻撃を仕掛けてきます。
　また、このモンスターは「二人で快感を共有するのが好き」という変な性癖を持っており、膣内を犯す時は自分の膣内にも尻尾を突っ込み、
　尻穴を犯す時は自分の尻穴にも尻尾を突っ込みセルフ中出しするのが大好きです。
　なんだこいつ。
　
Q.ランクはいくつ？

A.ユニコーンなどと同じ、4です。
　メスの獣なので女性系と動物系の両方に属しています。どちらかを指定すれば出て来やすくなるでしょう。

Q.何でこんなモンスターを？

A.お尻大好きなモンスターが欲しいと話題になっていた時に、「まだ子供なので、膣と間違えてお尻に突っ込んで子作りした気になっている子犬」モンスターを
　考えていたらいつの間にかこうなっていた。お尻の件はヴォジャノーイに先を越されたので断念。
　その分、コウノトリに負けないくらいカオス成分をマシマシにしてみた。
　だが私は謝らない。

Q.ドリフターってどんなモンスター？

A.触手に寄生された事で「空を漂流する程度の能力」を身につけた、マンボウです。まさかのマンボウです。
　攻撃は痛くも痒くもない通常攻撃と衣服攻撃しかしてきませんが、複数の特殊攻撃を仕掛けてきます。
　攻撃は「全体煙幕」「単体昏睡」「自分のHPを30%回復」「強制戦闘終了」の4種類で、いずれもかなりウザイです。
　他のモンスターと一緒に現れて妨害をしまくるポジションなので、邪魔をされる前に速攻で撃破してしまいましょう。
　ちなみにこのモンスターのHPは100しか有りませんが、「あらゆるダメージを1%に軽減する」特殊能力を持っています。
　その上で回復能力も持っていますので、気を付けて撃破してください。
　あれだけ触手にアヘらされていれば、マンボウだって防御力が上がるんですよ、やっぱ。
　
Q.ランクはいくつ？

A.最も湧きやすい、1です。
　能力はどう見てもランク1では有りません。
　しかし、NPCはこのモンスターを簡単に1ターンキルできるようです。
　どうやらNPCはダメージ軽減を無視して攻撃できるようで、ワラワラ湧かせないと恐らく遭遇する事すら難しいと考えてこのランクにしました。
　動物系と動物系の両方に属しています。どちらかを指定すれば出て来やすくなるでしょう。

Q.何でこんなモンスターを？

A.本スレの皆様、アイデアの提供ありがとうございました。
　HPの値だけ50ではなく100になっていますが、これは50じゃ低すぎたのが原因です。
　だが私は謝らない。
