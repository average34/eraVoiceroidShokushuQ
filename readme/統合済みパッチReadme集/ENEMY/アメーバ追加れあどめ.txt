触手宮101用アメーバ追加パッチ （製作者　喚く狂人）Ver1.0　初版

・概要
	敵としてアメーバというキャラを追加します。ランクは3、種類は8（丸呑み系）です。
	スライムに近い敵ということで、行動も多く流用しています。技IDの節約じゃよ……。
	ただ、バンシーの援軍に似た「自己増殖」という技を使います。援軍との差異は「（敵の登場枠がパンクしてない限り）必ず成功する」「敵が来るのではなく増える」「使用するとHPが減少する」ことです。
	ドリフターの「ひなたぼっこ」相当の技もあるので、確殺取れないとつらいです。

・敵の説明
	アメーバはスライムやブロブと似ていますが、生殖の形態において異なります。
	スライムの類は幻想少女の肉体を苗床として生殖を行いますが、アメーバは単独で、分裂して増えることができます。
	しかし、そうであっても少女が性感を得て垂れ流す様々な汁はアメーバにとって重要な栄養源であり、触手宮の怪物達の例にもれず、少女達を見逃すようなことはありません。
	アメーバ最大の特徴は驚異的な回復力です。些細な傷は、アメーバにとって無いも同然で、あっという間に元通りになってしまいます。先述した「分裂して増える」というのも、この回復力の応用です。
	見かけたなら、増えられる前に一気に畳み掛けてしまうか、それが出来ないなら煙球で逃げたほうが良いでしょう。中途半端な火力は、アメーバに通じません。
