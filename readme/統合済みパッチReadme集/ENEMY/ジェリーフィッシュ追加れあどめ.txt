触手宮101用ジェリーフィッシュ追加パッチ （製作者　喚く狂人）Ver1.0

・概要
	敵としてジェリーフィッシュというキャラを追加します。ランクは3、種類は16（海産物）です。
	HPは3000とランク３最低クラス（ランク１のワイヤーゴーストと同じ）ですが、炎でない攻撃を40％カットし、かつほぼ全ての攻撃に吸収効果が付いているため案外頑丈です。
	拘束後は拘束対象に対し、襲う/おねだりするのを強制する技を使ってきます。これを受けたキャラは行動がキャンセルされ、それぞれの行動をとってしまいます。こいつ単体しかいないときならともかく、集団戦でやられると厄介。
	なお、この強制は発情による自慰に優先されます。Revでいう「自慰し始める」をチョイスしたときみたいなものですよ、うん。

・敵の説明
	触手宮の中を浮かびながら漂流する、クラゲのような外見の触手がジェリーフィッシュです。
	ぶよぶよとしていて攻撃が通じにくく、こちらの体力を吸収するなど面倒な相手ですが、何より危険なのは頭にとりつかれた後です。
	一度ジェリーフィッシュにとりつかれると、身体の自由を奪われ、心にもないような淫らな行いを強制され、最後には苗床にされてしまいます。
	誰かが捕まったら、霊撃などで早く救出してあげましょう。
