●リビングバイン

地面をツタの群れがうごめいている……
リビングバインはこちらに気づくと、歓喜の雄叫びをあげるように
触手をいっせいに伸ばしはじめた！




●スカーレットバイン

天井から赤いツルのようなものが垂れている……
スカーレットバインはその血のように紅い触手を
鞭のように振りかざし襲いかかってきた！






;触手の基本にして完成形のバインさん
;弱キャラ一人だとこれにも勝てないのが泣けます。






●ワーム

膨れあがったミミズのような生き物がいる……
地中から出入りを繰り返していたワームは足音に気づいたのか
鎌首をもたげ突進してきた！





●サンドワーム

迷宮の床を削りながら、砂山が動いている……
地面を爆発させたかのように地中から這い出たサンドワームは
獲物を求め貪欲に動きだした！






;巨大ミミズ。人を丸呑みするぐらいだから、よほど大きいのでしょう
;倒すとダンゴムシみたいに丸くなるイメージがあります







●オクトパス

地上にかかわらず、タコのようなものが歩いている……
オクトパスは粘液に包まれた体をくねらせながら
迷宮の侵入者に向かって躍りかかった！



●クラーケン

部屋を埋め尽くさんばかりの巨大な影が見える……
丸太のような足を幾本も持つクラーケンは
その異様をもってこちらを見下ろしている！






;日本人の心。触手宮ではきっと非常食。
;足の数はタコでもイカでも種類によってバラバラだったりするようで








●イソツビ

壺が触手に覆われたような奇妙なものがいる……
イソツビは少女の下半身に狙いを定めると
跳ねるように体を伸ばしてきた！




●アクティニア

迷宮の壁に大きな花が咲いている……
花びらのような触手を広げていたアクティニアは
少女の愛蜜を吸いつくそうと壁を滑ってきた！







;幻想郷の人は海産系の生き物は見たことがあるのでしょうか？　外からきた図鑑とかで知ってそうな気もします。
;アクティニアは時計草のイメージです。外国ではイソギンチャクは花に例えられますね。






●スライム

名状しがたい液状の物体が這いずっている……
スライムは一度その動きを止めると
ごぼごぼと全身を泡立てながらこちらを威嚇してきた！



●ブロブ

肉塊のようなおぞましい物体が這いよってくる……
ブロブは新たな獲物を認めると体を急激に膨らませ
こちらを呑み込もうとしている！






;スライムと聞くとメガテンの可哀想な顔をしたあれを思い出します
;ブロブは液状化した肉のイメージで不透明なものを想像しました。……深く考えるとゲームシステム的にグロい。





●アルラウネ

大きな花の中に裸の女性が立っている……
アルラウネは半身である花を震わせながら、
淫靡な仕草でこちらを手招いている！



●ドリアード

木の傍に少年らしき人影が見える……
ドリアードは樹木と体を一体化させると
喜びに満ちた顔で狂ったように笑いだした！







;ユイチリのイメージということで迷ったのですが、ショタでもたまにはいいよね！（？）（少女に直してもおｋです）
;個人的には敵の中で一厄介






●シェイプシフター

●ドッペルゲンガー

〇〇の様子に少し違和感を覚える……。
××が話しかけると三日月のように口を開き、襲いかかってきた！






;文があると思いっきりバレバレなので無しのほうがよいですね……初遭遇時だけヒントとか？
;こういった怪奇系の敵は大好物です






●バイオプラント


××がその部屋に踏む入ると異様な雰囲気を感じとった……
部屋の中央には巨大な球根のような触手がそびえ立っている
カタマリの中心には薄い膜を張った液状のなにかが、内側にうっすらと人影らしきものが見える……


××が固唾を飲んでいると、カタマリの横穴から何かが這い出てきた
バイオプラントは新たな触手を次々と生み出している！




;ボス格なので、ややテキスト多めに
;苗床になっている少女はヘブン状態と脳内変換。







●ガーディアン

壁に沿って移動していた××はふと風が頬に当たるのを感じた……
迷宮の端に、他とは違った雰囲気の通路が見える
もしかしたらここから脱出できるかもしれない！

××が通路を進もうとすると、突然その入口が震えだした！
何かが奥から転がってきた事をを察した××は間一髪、飛び退る！

振動と共に出口を埋めたそれはあまりにも巨大な眼球だった
それがグルリと辺りを見渡すと、散らばっていた触手の残骸が寄り集まっていく……
無数の触手の鎧をまとった巨大な目玉はそのままこちらを睥睨してきた！

ガーディアンは迷宮からの脱出を許そうとしない！





;「このロリコンどもめ！」……スイマセン。とりあえず敵の姿解説テキスト通りの触手+目玉という感じに
;ボス格+脱出手段に直結するのでやや描写が多いですが、くどいかもしれません。
