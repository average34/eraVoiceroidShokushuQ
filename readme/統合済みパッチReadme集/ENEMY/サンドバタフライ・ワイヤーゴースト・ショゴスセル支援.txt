これから

　蝶の翅を持つ女性型の魔物　舌と脇腹、股間が触手化している。
　蝶の翅は砂や埃を巻き上げて獲物の視界を奪うほか、服を犯す鱗粉をまく。
　脇腹の触手は普段サンドバタフライの弱い部分を覆うように巻き付いている。
獲物を捕らえるのに用いるほか、触手として一番器用なので獲物の乳腺開発を行うことができる。
　舌の触手は獲物の口内を犯すことにのみ用いる。
花の蜜の代わりに獲物の体液をすする魔物だが、特に唾液が好物らしい。
そのため獲物を捕らえている間は舌で口辱し続ける習性がある。
　股間からは卵を獲物の膣内に産み付けるほか、口のように体液をすする機能もある。
卵は全て掻きだしても精液にまみれているため妊娠する可能性がある。

なお、キスはかなり得意らしく、戦闘力の高いものでもつい口の蜜を吸われすぎることがあるので注意が必要である。

これから
