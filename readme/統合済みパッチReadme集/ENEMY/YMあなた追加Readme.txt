触手宮101 YMあなた追加パッチ　プロト版

このパッチは触手宮にYMあなたを敵キャラとペットとして追加します。

あなたは激怒した。必ず、かの娘どもを横取りする触手宮を除かなければならぬと決意した。
あなたにはゲームバランスがわからぬ。あなたは、里の調教人である。穴を掘り、
娘達と遊んで暮らしてきた。けれども横取りに対しては、人一倍敏感であった。（略）

しかし本来の目的を忘れ、あなたは触手宮の妙な心地よさから居着いてしまっています。
果たしてあなたが触手宮を脱出できる日は来るのだろうか……？

ランクは7です。強さは人間の限界を勘案してるのでランクほどではないかも知れません。
ランクを下げるとYMあなたがうじゃうじゃいる迷宮になるので止めた方がいいと思います。
というか本当は専用の触手宮乱入イベント入れたかったんだけど作者にはLvが足りないんだ…。


YMあなたらしさがうまく表現できていません、あしからず。
YMあなたのくせに行動パターンが少ないです、あしからず。

後者に関しては可能であればアップデートで対応していきたいと思っています。

ペットパッチに暫定対応しました。
ペットになるともれなくあなたにツタと触手が生える暫定仕様です、あしからず。

確認していませんが、デバッグコマンドで召還しない限り
恐らく接頭語パッチの影響で「ひどく、淫乱なYMあなた」とかそんな名前に
なるかと思います、あしからず。
