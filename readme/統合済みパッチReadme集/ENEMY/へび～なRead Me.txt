特徴

共通
　名前の通り、蛇です。姿としては蛇の体に二本の体と同じ程度の太さの触手が生えています。
名前の「Ｃ」は…ぱっと見三つ首に見える⇒ケルベロス(Cerberus)　と　クリトリス好き　の
二つから。
　彼らの攻撃の特徴を一言でいえば「生きる集尿器兼利尿剤」。尿意を追加して吸引する、という
マッチポンプをします。吸引で回復、NPに応じたEPダメ増加、絶頂＆大量吸引で戦闘素質微減、
といったことをしてきます。
　ちなみに寄生攻撃は一切持っていません。ＶとＡはＶＡ複合技が二つほど、Ｂは拘束時専用一つに
衣服攻撃に一つになっています。
　また触手に寄生されたせいで全身から粘液を出すことが出来、炎系に若干の耐性を持っています。
過酷な環境に適用できるように強い生命力を持っており、自然治癒力が高めです。
　なに、蛇ならラミアがいる？　蛇というよりは女じゃないですかあれは。


Ｃワンプ　ランク２　動物

　速度と命中率が結構高めでありますが、戦闘素質はリビングバインにも劣り、HPも決して高くない
ので多分苦戦はしないです。かなり細いので遠距離攻撃（弾幕・消耗武器）を当て切れず、ちょっと
だけ耐性を持っているが、近距離攻撃（非消耗武器）は当たればちゃんと当たるので等倍。


Ｃアナコンダ　Ｃワンプから進化　ランク6　動物

　進化すると打って変わって速度と命中率が低めでありますが、HPと戦闘素質はフコウノトリの
90%ほどあります。なので急所狙いやカウンターを多用すれば比較的簡単に勝てるはずですが、
もしも万が一このような手段もなく、一人で対峙すれば苦戦を強いられるでしょう。
　より多くの粘液を分泌出来るようになっており、炎系への耐性がさらに少し上がっているが、
体が太くなっているので遠距離攻撃であってもしっかりと当てられて等倍、近距離攻撃なら三割
増し。…耐性の関係でフコウノトリよりもタフになってるかもしれませんが急所狙いが機能する
のでまだ戦いやすいはず…

概要（図鑑用）
========================================================================================
Ｃワンプ HP3000 戦闘素質40 速度100 命中135 ランク２　動物

　ごく一般的な蛇が触手に寄生されたもの。
自身の体と同程度の太さの触手が二本生えており、影を見ると首が三つあるように見える。
　触手に寄生された影響で生態が大きく変化している。
　まず、元来蛇は肉食であるが、Ｃワンプは動物から排出される液体全般を糧としている。
特に排出される機会の多い「尿」を積極的に狙い、排泄器官に直接食い付いて吸引したり、
排泄を促す特殊な液体を注入したりする。この特殊な液体には微弱な催淫成分が含まれている。
　また全身から微弱な催淫作用を持つ粘液を分泌することが出来、これにより乾燥や熱から
身を守ることが出来るようになっている。
　体が細長く、瞬発力もあるので弾幕等を放っても思ったようにダメージを与えられないが、
近づいて攻撃を当てる分にはしっかりとダメージを与えられるだろう。
　生命力が高く、中途半端にダメージを与えても一日あればおおよそ治り切ってしまうが、
幸い体は大きくないので倒し切ることはさほど難しくはない。
　ちなみに異種族間での生殖活動をほとんど行わない。あと催淫作用以外の毒は持っていない。
========================================================================================
Ｃアナコンダ HP18000 戦闘素質180 速度55 命中65 ランク６　動物

　Ｃワンプが多くの栄養と寄生している触手の作用によって異常な進化をしたもの。
本来の種ではありえないほど体が太くなっている。
　この異常な進化により、体の動きは鈍くなってはいるが、力・体力・生命力が大幅に強化され、
一度捕まると独力での脱出は困難になる。そのためこれと対峙する場合、複数人で急所を狙ったり
カウンターを多用するようにするとよい。
　全身から分泌される粘液の量が多くなっており、熱さにより強くなっている。しかし、体が
大きく、動きも鈍くなっているため弾幕等の遠距離攻撃手段でも当て切ることが出来、近接攻撃
であればより効率の良いところに攻撃できる。
　相変わらず異種族間での生殖活動をほとんど行わず、催淫作用以外の毒を持たない。
