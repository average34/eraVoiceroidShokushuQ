■■ 東方触手宮 スコールバイン攻撃手段追加パッチ

	生命力と胸の開発特化の敵らしいので、豊乳化する攻撃とB感度を上げる攻撃を追加してみました。
	触手特化系エロゲで良く見る、おっぱいを直接大きくする触手攻撃を実装してみたかっただけです。
	ペットにすればいつでもおっぱいおっきくできるね！

・具体的な変更点
	拘束時の性攻撃(強)で766番を使用。ペット時でも使用するように。
	TFLAG:98によりランダムで効果が分岐。2/3で豊乳化、1/3でB感度増大になります。
	豊乳化のコンフィグ制限(FLAG:521)が影響します。
	FLAG:521が1(豊乳化OFF)の時、またはFLAG:521が3(男の娘のみOFF)かつ攻撃対象が男の娘の時は、必ずB感度増大攻撃になります。
	豊乳化には妊娠出産の処理を流用しています。

	修正・改造など、御自由にどうぞ。


・使用している変数番号
	FIX_ENEMY_ACTION_766
	SYSTEM_MESSAGE_4,14 (性攻撃　発動＆命中)
	766　スコールバイン　B　乳肥大化 or B感度増大
