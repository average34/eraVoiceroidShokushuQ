触手宮101用敵バリエーション増加パッチ （製作者　喚く狂人）Ver1.0　初版

・概要
	「The Elder Scrolls IV:Oblivion」のMOD、「Martigen's Monster Mod」の「No two creatures will ever be the same.」機能に影響を受けて作成したパッチです。
	敵に接頭語が付くようになり、それに応じたステータス補正が入るようになります。各ステータスに強化と弱化両方の接頭語があり、付く数やそのランクは難易度により変動します。
	この変更により、接頭語に応じた作戦を立てる必要が出るようになり、総合的には難化します。
	なお、接頭語が付くタイミングは敵が生成される瞬間なので、既存のセーブデータでプレイする場合、既に居る敵には変化がありません。新しくスポーンするのを待ってください。
	高難易度のニューゲームで初めていただくとパッチの効果がよくわかるかと思います。
	接頭語の情報はDA等に記録されていないので、他パッチなどから判断したい場合は直接SAVESTRを見るか、@CREATE_ENEMY_RANDOMIZEを弄ってください。

・危ない接頭語
	「元気な」「よく動く」は、行動回数を増加させる接頭語です。単純に二倍三倍強くなります。
	「強い」「凶悪な」「恐るべき」「神の如き」は、体力・攻撃力・速度・耐性をやんわり上昇させる接頭語です。他の接頭語と組み合わさった場合、相乗効果で恐ろしいことになります。
