通常時は基本的に拘束攻撃と性攻撃、そして恐怖の全体妊娠攻撃、「空気妊娠」を行います
拘束時はひたすら妊娠攻撃をしかけ、邪魔する相手には羽根飛ばしで通常攻撃を行います
ただし拘束した相手が既に妊娠していると分かった時は即座に解放する上に、戦意喪失時には通常よりも高確率で解放してくれます
注意点ですが、全員が妊娠しているか射精されすぎて妊娠確定の時には基本的に離してくれません
喪失していても妊娠していても泣いても喚いても離してくれなくなります
これはひどい

自動販売機で娘を増やす作業に飽きた時は、ぜひコウノトリを活用しましょう
