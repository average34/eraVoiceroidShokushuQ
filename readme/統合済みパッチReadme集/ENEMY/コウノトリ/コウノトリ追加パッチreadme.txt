・本パッチは東方触手宮V0.071＋まとめパッチに新モンスター「コウノトリ」を追加するパッチです。
　概要は以下のQ&Aを御覧ください。

・Q&A

Q.コウノトリってどんなモンスター？

A.とある鳥と触手が共生する事で生まれたモンスターです。
　体長2～3mはある大型の鳥の姿をしており、体のところどころから触手が伸びています。
　このモンスターの目的は「雌を孕ませる」事のみで、雌を見つけると一気に急降下しつつ足でガッシリと相手を捕らえ、
　羽根を使って作った即席の巣の中で、男根状に変化させた触手を使ってひたすら交尾と言う名の種付けを行います。
　その名に恥じぬ、正に子供を運ぶ「コウノトリ」です。
　一見そこまで厄介なモンスターでは無いように見えますが、1回の妊娠攻撃で注がれる精液の量は他のモンスターの数倍～数十倍にもなる上に、
　「妊娠攻撃しかしない＝MPが直接ゴリゴリ削られる」、「直撃＝まず即死の十連妊娠攻撃がある」と地味に強力な攻撃を仕掛けてきます。
　しかも「通常時は高確率で拘束攻撃を仕掛けてくる」ため、他のモンスターと一緒に現れた際には最悪の障害物と化す可能性があります。
　体力はそこまで多くない上に通常攻撃に弱いので、こいつと出くわした時は真っ先に倒しに行きましょう。
　なお、このモンスターは他には無い、「拘束攻撃を仕掛けた相手が既に妊娠していると即座に開放する」紳士的な特徴を持っています。
　自動販売機に頼らずに娘が欲しいそこのアナタ、子供はコウノトリに頼むと便利ですよ？
　

Q.何匹ぐらい出てくる？

A.初期配置で2体出てきます。
　それ以降はそこそこ早いペースでポップするはずです。


Q.何でこんなモンスターを？

A.ちょっとぐらい変な挙動をするモンスターがいたっていいじゃないかと思った。
　反省？俺は今猛烈にてんこから優しく叱られたいから、今はまだしないのさ…
