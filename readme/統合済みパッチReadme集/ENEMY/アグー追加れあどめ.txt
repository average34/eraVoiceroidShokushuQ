触手宮101用アグー追加パッチ （製作者　喚く狂人）Ver1.0　初版

・概要
	敵としてアグーというキャラを追加します。ランクは3、種類は4（動物系）です。お住まいの地域やご職業によってはお察しいただけるかもしれませんが、豚です。
	犬、馬、鳥、狐、カエル――獣姦要素がこれだけ充実しておいて豚が居ねぇってどういうことだ！　無いんやったら作るぞ！　という憤りがいつの間にかERBという結晶を形作っていました。
	特殊行動「悪臭の息」にてバッドステート付与・デバフを行いまくってくる邪魔臭いタイプの敵です。ダメージはでかくないので、一体相手ならゴリ押せば終わりますが。
	敵行動としてダメージ低下デバフを作成したので、どちらかといえばそちらが目玉かもしれない。

・敵の説明
	ハウンドやユニコーンのように、この迷宮には様々な動物……のような姿をした触手――がいますが、アグーもその一種で、豚のような姿をしています。
	豚舎からは独特のいやな臭いがしますがアグーも同様で、全身からのぼる悪臭が彼らの武器です。というのも、彼らの酷い臭いの正体は毒であり、相手にさまざまな被害を与えるのです。
	複数人で一体を攻撃するのであれば、体力も力も速さも無いアグーは楽な部類に属しますが、彼らが別の強力な敵と組んで現れたときは厄介なことになるでしょう。

・注意事項
	戦闘周りの処理を弄る。
	CFLAG:303を戦闘用ダメージ低下フラグとして追加。（りばーすのものに似せた仕様で、基礎ダメージ値を半減させてしまうので、特殊効果36を使う技を使用する場合はあまり使用率を上げ過ぎないこと）
	ちょくちょくりばーす用のものを流用。
