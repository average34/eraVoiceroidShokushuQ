敵「ホフゴブリン」追加パッチver1.1　by喚く狂人

・概要
　探索イベント「ガチャガチャ」で特定の条件を満たすと、敵「ホフゴブリン」が湧くようになります。ランク2、分類は32（その他）です。
　ランク2でHP2000、戦闘素質30と悲しくなるステータスですが、速度は120あります。さらに2回行動です。しかも誰も拘束していない場合、拘束されていないキャラを最優先で狙ってきます。
　とはいえ、使用する技もそこまで強くないので、大した敵ではないです。
　エロ的な話をすると、一体の枠で数体ぶん居ることになってる変な奴で、ゴブ姦、輪姦、見物プレイあたりのニーズに応えています。苦手なひとはイベント回避してれば出てこない！　親切！（？）

・敵の説明
　触手の占める迷宮において、上手く触手と共生しているのがホフゴブリンである。普段は触手達の世話などをして暮らしている。迷宮の隠微な空気のせいか、その一物は常に勃起しており、自らの背丈にも劣らぬサイズになっている。
　幻想郷にも存在する彼らホフゴブリンは、本来ならば外見こそ醜いものの善良な働き者である。だが、触手達との生活の結果なのか堕落してしまったようだ。
　この迷宮のホフゴブリンは抜け目なく、ずる賢く、通常は迷宮内の穴に潜んでいるため、幻想少女たちに対して直接その身を晒すことはない。
　少女たちに対してはまずは迷宮の触手たちを差し向け、彼女らが触手に屈したところで、その「おこぼれ」に預かるのだ。
　そんな彼らも、自分たちの棲家を壊されると怒り狂う。持ち前の素早さを生かし、集団で襲い掛かり、舐めた真似をした雌に己のモノで制裁を加えようとするのだ。
　そんな時ですら単独では動かないのが、彼らのずるさを表しているのかもしれない。
