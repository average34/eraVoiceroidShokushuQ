ツタの拘束、通常の戦闘力はそれほど恐ろしくはないが体内に捕らえられると一転、恐ろしい攻撃が放たれる
だが捕らえていない獲物に対してはそれほど強力な攻撃ができないので複数で襲い掛かれば誰かを犠牲に簡単に倒せる。
エサから生命力を充分に吸い上げた固体はマンイーター(未実装)と呼ばれ能動的に獲物を求め彷徨うようになる(かも)
凶暴性と知能が増し(たらいいな)、体力の少ない獲物を執拗に襲い、力尽きたところで触手で襲い掛かる(予定)
だが動きの緩慢さは相変わらずで触手で拘束しきった獲物以外を飲み込むことはできない(はず)

おまけ(R-18 ハードくらいの表現)
神隠し異変が解決した後も触手の迷宮から逃げ出した魔物の被害はぽつぽつと報告されていた。
だが、時が立つにつれて魔物の残党は次々と退治されていき異変があったことすら人々は忘れかけていた。
人々が忘れかけているようなことを妖精が覚えているわけもない。
だから大妖精は見慣れないツタ植物を前にしても何も警戒はしなかったし、
ツタが動いたことに気づいてものんきに「珍しい！動く植物なんだ！」と好奇心に胸を躍らせただけであった。
彼女の頭の中はツタから生えている可愛らしい花と鼻をくすぐる甘い香りと好奇心でいっぱいで、
どうやってツタから花を摘むかなんて考えながらツタの近くにしゃがみこもうとして何かを踏んづけて後ろに転んでしまった。
背中を強く打った痛みに悲鳴を上げ起き上がろうとしたその時、足首を撫でるくすぐったい感覚に大妖精の全身に鳥肌がたった。
びっくりして大妖精が足首に目をやるとツタが巻きつき上へ上へと蛇のようにはいあがっている…
身の危険を感じた大妖精はそれを引きがそうと悲鳴をあげながら必死に脚をばたつかせるがそうこうしている間に今度は腕に、体に、腰に、ツタはみるみるうちに絡み付いてきた。
大妖精は一度大きく深呼吸をし、落ち着いてまずは右手で左腕に巻きついているツタをはがそうとツタを掴む。
その間に更に一本の触手が右足に巻きついた。
左腕を振り回し、なんとか絡みついたツタを振りほどいた。
その間に更に二本のツタが両足に巻きつき、彼女の股間を舐めるようにすりあげた。
たまらず恥ずかしさと感じたことのない感覚に自由になった両手で股間を守る大妖精は次第に自分の体が引きずられていることに少しも気づいてはいない。
彼女が泣き叫び助けを求め殆ど平面に近い胸を撫でられ慌てて胸を庇いそして再び無防備になった股間をつつかれ今度は股間を慌てて守り引きずられる。
叫ぼうと口を大きく開けた大妖精の顔面にツタの先端から白濁液を吐きかけられ、空気と一緒に器官に入り込んだ液体の苦しさにケホケホと苦しげなせきを繰り返し、その隙にさらにツタがまとわりつく。
大妖精の涙でぐしゃぐしゃになった顔が凍りついたのはすぐ近くに歯のない巨大な口が彼女を招き入れるようにゆっくりと開閉していることに気づいた時だった。
自分が置かれている状況が自分が思っている以上に危険なことに気づき、悲鳴すら上げることができずガチガチと歯を鳴らして震える大妖精。
だが、その植物は無慈悲に彼女の体を口へと運ぶとあっさりと小さな大妖精の体を引きずり込んだ。
脚のなだらかなラインを舐めるように口がなぞりそのまま細いウエストまでずるりと飲み込まれる。
下半身の生暖かく湿った不快な感覚に大妖精は再び思い出したように悲鳴をあげようとし、口内に残っていた白濁液を吸い込み再び咽る。
もう一度口が大きく開くと同時に重力とツタと肉壁の動きのより、大妖精は完全に呑み込まれてしまった。

彼女を迎え入れたのは幸いなことに消化液ではなかった。
内臓のベッドに磔られた彼女の全身に絡みついたツタは身動きを許さず、彼女の秘裂には先ほどから何度も味わうように下着の上からごつごつとした触手が擦り付けられていた。
彼女の体の柔らかさを楽しむように触手は全身をもみしだき、特に胸には先端が花のようになった触手に吸い付かれていた。
完全に抵抗することを諦めた大妖精は怯えた瞳で自分の股間に擦り付けられる触手を見つめ、何をされるか分からない恐怖に身を硬くするだけだった。
突然背中を駆け上がった得体の知れない感覚に大妖精は体をぴくんと反応させた。
どうやら、下着に穴が開き直接触れられたたらしいと大妖精は察すると恥ずかしさで耳まで一気に赤くなる。
そして次の瞬間に、大妖精は激痛に真っ青になった。
ブチブチと体が脚と脚の間からちぎれる音が激痛で真っ白な頭の中で木霊する。
不幸なことにそれで大妖精は一回休みになることは出来ず、ゴリゴリと膣壁をこすられる痛みにさらされる羽目になった。
痛みのあまり気が狂ったかのように叫び続ける大妖精は膣壁をこするのは触手ではなく触手内を通っている卵であったことに気づいたのは、触手が引き抜かれた後にぽっこりと広がった自分の腹をみた時だった。

人里から離れた小さな野原に最近動く植物がたくさんいるらしいと妖精たちの間で噂が広がっていた。
ある者は見間違えだと、ある者は妖怪だと、ある者は新種だと言って面白半分に何人かはそれを見にいった。
そして見に行った者が一人残らず帰って来ていないことになど妖精たちは気づくことはなかった…
