・概要
	敵としてフレッシュゴーレムというキャラを追加します。ランクは5、種類は8（その他）です。
	本来は死肉で作るゴーレムなんですが、まあこれ触手宮だし触手ってことでいいよね。ちんこがでかい拡張プレイ要員。精液ボテ要素は結構あるのに、なんか拡張ってあんまり居ないよねー。……それが作成動機です。
	同ランク最高クラスのHPを誇るデカブツです。足おっっっそいけど。本来は死肉で作るゴーレムなんですが、まあこれ触手宮だし触手ってことでいいよね

・敵の説明
	大小あまたの触手がより合わさり、石像ならぬ触手像とでもいうべき姿になったのがフレッシュゴーレムです。
	ちょっとした家ほどの大きさもあるその巨大な体躯は、それだけでも対峙することを躊躇わせるほど圧倒的なものです。
	さらに、外見から想像できるとおりの怪力で、か弱い幻想少女たちでは一度捕まると逃げることは困難です。
	幸いにも、多数の弱い触手のより集め、いわば烏合の衆であるため、統率がいまいちであり、動きはかなり鈍重です。
	落ち着いて相手の動きを見る冷静さがあれば、強力な攻撃も躱すことは不可能ではありません。
