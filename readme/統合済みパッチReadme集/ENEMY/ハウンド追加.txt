3行まとめ
犬の群れに擬態した婬虫です
弱らせて動きが鈍くなったら捕まえて四つん這いにします
後ろから近づいて2つの穴を堪能します

迷宮に大量にいる淫虫が犬の群れに擬態した魔物です
獲物を弱らせることを優先したり足を狙ったり連携攻撃する知能があります
素早く命中も高いですが耐久や戦闘素質は同ランクでは平均ほどです
味方を包囲したり（効果は未実装）身体をほどいて床に潜ったりします

HPが低くなった相手は四肢を拘束し抵抗できないまま襲われてしまいます
素早く複数いるので近接武器は当たりにくいですが火と弾幕には弱め
