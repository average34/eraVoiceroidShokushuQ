設定とか

バンシー
触手に寄生された一般人です。
普段は自分の体で触手を鎮めていますが他の女性を見かけると襲いかかります。
自我は残っており、いつもすすり泣いているためバンシーと呼ばれます。
比較的改心しやすいですが脱出する気はなく、その実力もなく触手で自分を拘束し自慰を続けます。
改心しても身体の開発が進んでいる上に戦闘力も弱いため、拘束状態で放置して母乳搾りや性欲発散に使うか、
あるいは囮にするくらいしか使い道がないのですが……
むしろ放っておくと敵に苗床にされて産まされ続けるか、バイオプラント化するので扱いには困るかもしれません。


スキュラ
下半身がイカやタコなどの触手な女性です。
クラーケン等と同様ヌルヌルとした触手で胸と陰核を中心に狙います。
高めの耐久と胸狙いからの母乳吸いがあるのでかなりタフです。
欠点として弱いものをいたぶるのが大好きな点があります。
戦意を喪失していても離さずに襲い続けるため、誰かを囮にすれば戦うのも逃げるのも楽でしょう。
一人の時に出会ってしまうとそれはそれで楽しいことになりますが……
