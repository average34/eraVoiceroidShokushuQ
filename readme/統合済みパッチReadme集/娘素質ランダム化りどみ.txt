これは触手宮1020用の、娘の素質がランダム化するパッチです。
と言ってもよく変わるのはだいたいTALENT26番までの基本的な性格的な部分と、痛み関連・濡れ関連だけですが。
また、一部の性格は強気や弱気の度合いに偏りが出るようにしてあります。
具体的に言うと弱気なねっけつかんや強気なひっこみじあんなどが出にくくなりました。
そのせいでテストプレイしてる時に普通な○○ばっかり出てどっかミスったかと不安になった。
……まぁ処理の関係上、普通な○○が多くなるのは当たり前なんだけど。
ついでにシェイプシフターな娘やドッペルゲンガーな娘にほんのり素質追加。

あとテストプレイ時に壁尻スライムから謎のエラーが出たため同梱しておきます。
