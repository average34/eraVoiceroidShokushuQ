------------------------------------------------------------------------
■ 東方触手宮1.04.5 キャラ操作(削除、入れ替え、ほか)パッチ ■
------------------------------------------------------------------------

仲間にしなかった娘のキャラがそのまま残ってると、
キャラID数がいっぱいになった時に色々面倒な事になる気がしたので、キャラ削除できるようにしてみました。
ほか色々な部分でデバッグ操作しやすい様になったと思います。
(デバッグモード)

--------
・キャラ削除について
PTメンバは削除したらエラー吐くので、削除できないようにしてあります。
削除したい場合は仲間から外した後にしてください。
主人公のキャラは、キャラ入れ替えでPTメンバから外した後なら削除できます。



・キャラ入れ替えについて、
名前や各種ステータス、CFLAG、EXPなどのキャラ毎の値はまるごと入れ替わります
PTメンバフラグは、そのIDに対して紐付けされているので入れ替えの対象にはなりません。

敵パラメータのDA:48から参照されているID(○○の娘)は、入れ替え対象に選択できないようにしてあります。
(改心前に入れ替えてしまうと、改心後仲間に引き入れた際におかしくなるので)
それ以外のキャラなら、主人公でもPTメンバでも入れ替え可能です。

魔法の拘束具を付けて敵に拘束されているキャラを、PTメンバと入れ替えたりする時は注意。
特に、拘束されているキャラを主人公と入れ替えてデバッグモードを終えた場合、
その場から身動きが取れなくなるので、メイン画面に戻れず時間だけが過ぎて行きます。
ただし、敵と遭遇して勝てばメイン画面に戻れます。
敵から逃げてしまうと、仲間だけが逃げて主人公が置き去りにされるので、また身動きが取れない状態になります。
主人公に対して放置プレイさせたい場合はどうぞ。



・○○の娘のキャラIDについて、
いくつかの状態(改心前 or 改心後に引き込まなかった or ザコ一掃でゴミデータ化した)に当てはまる場合
現在地が59になるため「隔離部屋」と表示され、改心前の状態なら敵IDも一緒に表示されます。
改心前の状態のキャラIDを削除した場合は、該当する敵キャラも同時に削除されます。



・その他
今まではザコ敵の全削除処理しか無かったので、
ザコ敵の選択削除処理もできるようにしました。
アイテム操作画面も3列表示にして改行回数を減らしました。
