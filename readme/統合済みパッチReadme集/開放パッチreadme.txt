触手宮0.71用捕食中キャラ開放パッチ v0.2

●使用フラグ●
CFLAG:45


●使い方●
触手宮パッチまとめを適用した上でこのパッチを上書きしてください


●内容●
またの名を犯り捨てパッチ
雑魚敵に捕食されている時に一定の確率で孕ませて開放してくれるようになります
また、既に妊娠している場合も開放してくれます
コンフィグでON/OFFの切り替えができるので、嫌な場合OFFにしてください
主人公が開放された場合MP1で復活します
したがって多少難易度が低下します(多分)


…というのは建前で
実際はキャラがボロボロの状態で逃げ回ってる様を妄想して楽しむためのパッチです

実際、開放されても敵が集まってて逃げられなかったり
主人公以外は戦闘不能から復帰しないので輪姦状態だったり
開放されるけど妊娠したり
そもそも捕食される時点でヤバい状況だったりと
どのくらい難易度低下の効果があるかは不明
