1. 何このパッチ？
	触手宮ふたなりの状態異常発生パッチ…の勝手に追加パッチです。それだけ。

2. どうやって使うの？
	東方触手宮v0.07.1用ふたなりの状態異常発生パッチの後にこのパッチを上書きします。
	必要に応じてFUTANARI_ADDITION.ERB中の@FUTANARI_PERCENTのLOCAL:1 = ○○を書き換えて自分の好みに調整できます。

3. 何が出来るの？
	主人公は自発的には行動を起こせません。

4. 何をされるの？
	味方NPCが確率でふたなり化します。(パーティ内の味方は含まれません)
	ふたなり化した味方の好感度が高い状態で見張りを頼んで寝ると……
	それとは別に好感度が高いと寝てるのを見かけて……
	Ver.0.08からはアイテムをねだる時にパーティ以外で同室に居る味方が一人だと……

5. メリットは？
	ありません。作者の俺得なだけです。
	…と思いきやVer.0.03からメリットができました。
	好感度が非常に高いNPCに襲わせる（？）と精神力の回復を意図的に行えます。
	あとVer.0.08からアイテムをねだりやすくなりました。

6. デメリットは？
	ひどいくらいにあります。
	ふたなり化して射精するとHPがガリガリ削られていきます。故にNPCのHPがマッハで消えます。不死身じゃないと辛い。
	仲の悪い人に見張りを頼んでも逃げられる、仲が良すぎると……。どうしろと。
	ちなみに設定次第で仲が悪くても襲ってきます。その場合、相手を辱める為だけにって事になりますね。あれ、俺得だ。
	襲われると場合によってはMPががっつり減ります。寝て回復するはずなのに。
	アイテムをねだる時に好感度が低いと…これもまたデメリット。

7. 注意とかある？
	触手宮ふたなりの状態異常発生パッチ以外とのパッチとは整合性を全く取っていないので共存できない可能性があります。

8. 最後に？
	ステータス変化とかは適当に設定しています。
	各種ダメージ調整を投げ出し気味。
	寝てる最中なので直後に睡眠回復があってダメージを受けても無効化される事が多々あるかと思いきや、
	好感度が高いとMP全快から一気に削られて喪失する事態も発生。
	(テストプレイ時の事なので極僅かに残るように修正してあります。あとVer.0.03から非常に高い場合は分岐)
	Ver.0.04より触装云々の追加を行いましたがバランス等は考慮していません。
	陵辱色強めに体力が尽きようが戦意喪失しようがひたすら絡ませ続けます。
	解放条件は触手が尽きるかもしくは……まぁ、母体に無理はさせられないですよね。
	あと攻め側の体力が尽きた時に確率で気絶するようにしました。
	(ほんの数時間で発情した味方が背後から抱きつく云々と並行しながら作ったので地の文とかはご愛嬌)
	勝手ながら主人公がふたなりにならないと嘆いていたようなのでこちらでも少し主人公がふたなりになるイベントを追加しました。
	例によって確率をいじれます。条件は外田氏。
	脳内妄想としては外田氏なんてするやつに寄生していても生殖が見込めないとして別の人を宿主にする…みたいな。
	Ver.0.08のイベントは元々の地の文「値踏みするように肢体を上から下まで眺めて」を見て妄想せざるを得ない展開です。俺は悪くない！
	使い道はあまりないような気もしますがふたなり伝染のイベントを任意のキャラにできるようにしました。
	CALL FUTANARI_MOVE(伝染元キャラ, 伝染先キャラ, 0or1)でできます。但し素質ふたなりや、伝染先が既にふたなりの場合は伝染しません。
	0は同意無し、1は同意有りの口上が表示されるようになっているので表示しない場合は2以上を指定すれば表示されません。
	改変等はして頂いて結構です。むしろ良くなるならどんどんしてください。	

	主人公がやられるの大好ぶｔ…
