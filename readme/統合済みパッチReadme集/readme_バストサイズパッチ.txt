・パッチについて
このパッチは絶壁→貧乳や巨乳→爆乳といった胸素質の変更を容易に行える関数を追加するパッチです。
また、サイズ変更時の口上呼び出しも追加します。


・主な使用ファイル
BUST_バストサイズ処理.ERB
SYSTEM_MESSAGE_510_バストサイズ変化.ERB


;=================================================
;バストサイズ取得　※式中関数
;ARG:0 対象キャラID
;RESULT:0 バストサイズ（-2:絶壁　-1:貧乳　0:普乳　1:巨乳　2:爆乳）
;=================================================
@GET_BUST(ARG:0)


;=================================================
;バストサイズ変更
;ARG:0 対象キャラID
;ARG:1 何段階変化させるか(正数:バストアップ　負数:バストダウン)
;ARG:2 理由
;ARG:3 理由の補足
;RESULT:0 変更後のバストサイズ（-2:絶壁　-1:貧乳　0:普乳　1:巨乳　2:爆乳）
;=================================================
@ADD_BUST(ARG:0,ARG:1,ARG:2,ARG:3)


;=================================================
;バストサイズ設定
;ARG:0 対象キャラID
;ARG:1 バストサイズ（-2:絶壁　-1:貧乳　0:貧乳　1:巨乳　2:爆乳）
;ARG:2 理由
;ARG:3 理由の補足
;RESULT:0 変更後のバストサイズ（-2:絶壁　-1:貧乳　0:普乳　1:巨乳　2:爆乳）
;=================================================
@SET_BUST(ARG:0,ARG:1,ARG:2,ARG:3)


;=================================================
;バストサイズ変化時のメッセージ（現状は対象キャラのみの呼び出し）
;ARG:0 メッセージID（510固定）※使わない
;ARG:1 対象キャラID
;ARGS:0 対象キャラのCALLNAME
;ARGS:1 未定義
;ARG:2 変化前のサイズ（-2:絶壁　-1:貧乳　0:普乳　1:巨乳　2:爆乳）
;ARG:3 変化後のサイズ（-2:絶壁　-1:貧乳　0:普乳　1:巨乳　2:爆乳）
;ARG:4 理由（0:その他 1:アイテムを使った 2:出産時豊乳化 3:自販機）
;ARG:5 理由の補足
;	ARG:4 == 1 の時(アイテムID)
;ARG:6 未定義
;=================================================
@SYSTEM_MESSAGE_510(ARG:0, ARG:1, ARGS:0, ARGS:1, ARG:2, ARG:3, ARG:4, ARG:5, ARG:6)
