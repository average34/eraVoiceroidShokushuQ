●ドッペル判別
特定の条件下でドッペルゲンガーを見抜けるようになるパッチ。
条件を満たすと異常に気付いた内容の168の口上が出る。
条件は以下のとおり。

○PTにさとりがいる
　百発百中一発判別。ただし戦意喪失になっていると駄目、発情中だと50%に低下。
○主人公が気配察知・話術・危機回避MAX
　百発百中一発判別。ただし戦意喪失になっていると駄目、発情中だと33%に低下。
　ただこの条件を満たしていれば後述の条件も満たすので実質的にそれより上。
○PTに娘がいる
　25%で判別。キャラごとに見るので二人とも娘だと確率上昇。
○PTに話術・危機回避のいずれかがMAXのキャラがいる
　12.5%で判別。キャラごとに見るので複数いるとその分確率上昇。
