触手宮0.71まとめ用イベント追加パッチ詰めあわせ

●内容
拘束キャラ開放パッチ
自販機追加イベントパッチ
白濁泉追加イベントパッチ


●使い方
触手宮パッチまとめを適用した上でERBフォルダ内のファイルを上書きしてください
9月7日現在
ヒュプノパッチとの共存が可能です
尿意パッチと併用する際は尿意パッチを適用した上で尿意パッチ対応版のフォルダ内のファイルを上書きしてください


●説明
○拘束キャラ開放パッチ説明

使用フラグ
CFLAG:45

雑魚敵に捕食されている時に一定の確率で孕まされるかわりに開放してくれるようになります。
また、既に妊娠している場合も開放してくれます。
主人公が開放された場合MP１で復活します。
仲間の場合は開放した直後に同じ敵に連続で拘束されることはありませんが、
他の手の開いている敵がいる場合そいつに捕まってしまうのであまり効果はありませんが、多少難易度が低下します

…というのは建前で、
実際はボロボロの状態で逃げ回る様を妄想して楽しむためのパッチです。


○自販機追加イベントパッチ説明

自販機に新しいイベントを１つ追加します。
使用フラグはありません。

定番だと思ってたのにいつまで経っても追加されないから自分で追加。
地の文は期待しないでください。書き直し大歓迎です


○白濁泉追加イベントパッチ説明

白濁した泉に入ったときに起こるイベントを１つ追加します。
使用フラグはありません。
