■ 修正内容的な

戦闘中、催眠に掛かっているにも関わらず通常行動選択が出来る不具合を修正
※画面に表示されないだけで直接入力すると防御やアイテム使用が可能だった

リピート行動時、選択不能なコマンドをリピートする不具合を修正


■ ソースコード的な

通常時・催眠時のコマンド選択を一つの関数でやっていたので二つに分離しました
通常時・催眠時・拘束時の処理方法を統一
これで選択できないコマンドでリピートすることは無くなる…といいな

おねだりコマンド毎に入っていた部位ランダム化処理を
コマンドの対象選択(SELECT_COMMAND_TARGET)に移動
これで今後おねだりが増えても、同じ部位だけおねだりすることが無くなるはず
※今までの処理でも問題なく動いていたけれど、一応ミス防止のため
