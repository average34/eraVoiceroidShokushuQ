パッチの内容
１．パラメーター【NP】(尿意ポイント）の追加
非戦闘時、ターン経過ごとにNPがちょっとずつ溜まっていきます。【小柄体系】持ちだとたまりやすいです。膀胱が小さいから。
水を飲む系統のイベントでももちろん増加します。
NPが高いときに絶頂すると、確率でおもらしをしてしまいます。
２．おもらしの追加
そのままです。NPが高いときにおもらしをする可能性が出てきます。
おもらしをするとTPが割と多めに下がり、CPが3/4になり、状態異常[ノーパン」になります。パンツが駄目になったんです。
３．状態異常[ノーパン]の追加
同じくそのままです。ぱんつはいてません。
B以外の性攻撃が全裸の時と同じ扱いになります。リグルがズボンだなんて気にしちゃいけない。
今のバージョンだとC攻撃しかないんだけどね！
治すには全裸になるか、着替えセットを使うしかありません。
４．アイテム[携帯トイレ]の追加
おもらし以外でNPを0にする手段になります。普通に探索で拾えます。
５．コマンド[お花を摘む]の追加
上と同じくNPを0にすることができますが、以下の点が異なります
　①全員で行きます。女の子は集まってトイレに行きますもんね
　②同じ部屋に敵が居ると使えません。
　③NPが高いと、間に合わずおもらししてしまうこともあります。
　④部屋の汚染度が高いと、マイナスイベントが起こることがあります。
６．おねしょの追加
そのまま（ｒｙ　NPが高いときに寝るとおねしょをしてしまいます。

……とまぁ、結構システムの根幹に関わる場所もいじったりしてます。特に戦闘関連は「敵の攻撃」１つ１つに設定変更が及んでいます。
こうなると元の作者さんに本体に組み込んでもらうのも大きな手間になってしまうので、望むべくもありません。
というわけで、このパッチを導入する際の注意点は以下になります。

１．バージョンアップしたら基本的に使えなくなります。
２．バージョンアップされても、対応したパッチが出るとは限りません（作者は結構やる気の推移が激しいです）
３．本体に組み込まれる可能性はまずありません

以上を理解した上で使っていただけるとありがたいと思います。
