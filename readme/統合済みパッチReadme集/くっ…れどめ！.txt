　　　　　↓

・何が出来るの？
PRINT_VALUE_STR.ERB 256行目以降に手を加え
月経周期で割り出した排卵日＆危険日の判定結果をCFLAG:159に記憶させました。
それぞれ
CFLAG:159 == 0　　 →　　安全日、生理日、妊娠中等
CFLAG:159 == 1　　 →　　排卵日
CFLAG:159 == 2　　 →　　☆危☆険☆日☆

このパッチを当てる事で、口上側で「危険日」や「排卵日」の判定をさくっと分岐させる事ができます。
嫁に「妊娠だけは許してぇ～」
　　『いいや妊娠だけは確実にしてもらう』
とかの分岐を作ってあげたい口上作者にお勧めです。
(というかうｐ主が自分用に作ったような……)

・生理日は？
DAY % 30 == CFLAG:(ARG:0):150 + 1 だと生理の日だったりします
