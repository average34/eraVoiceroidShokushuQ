パンツ脱がしパッチ　りーどみー
[概要]
敵や服が増えてるし更新したほうがいいよね、って電波が来た。
後、今回、壁触手や床触手さんが罠扱い→敵Noが0→パンツ脱がしパッチ発動しない（CFLGA:20を見ていたため）と言う状況になったので若干行動定義にも手を加えました。（CFLAG:20を見ていたところをCFLAG:21も見るようにしただけ……なのでおそらく不具合は出ないと思いますが）



[導入の仕方]

[すぺしゃるさんくす]
