触手宮101用ランダムで発情したりするパッチ （製作者　喚く狂人）　初版

・概要
	歩いていると確率で発情したり自慰したり他のキャラを押し倒したりするようになるパッチです。
	発情はそこそこ発動しますが、自慰・押し倒しに関しては結構レアです。
	喪失になってるキャラは対象外です。
	コンフィグから有効化することで遊べます。

・注意
	FLAG:530使用（コンフィグ用）
	CFLAG:600使用（最後に発情したりした時間が入ってます）

・適用の仕方
	そのまま放り込んで上書きしてください。
