これは触手宮1017において設定をスムーズに進めるように改変したパッチです。
eratohoYMを参考に、プレイヤーの趣向ごとに初期設定・フィルターをまとめて設定できるようになりました。
なのでどちらかと言うと触手宮をプレイした経験が少ない人向けになります（今更ですが……）。
内容は以下の通り。

●初期設定
・基本セット（触手宮に初期から設定されていたモンスター及び機能のみオン）
・戦闘セット（戦闘関連の設定をオン、モンスターはフィルターなし）
・堕落セット（身体変化系を中心にオン、モンスターは初期モンスターと一部追加モンスター以外にフィルター）

●フィルター
・個別フィルター解除（個別フィルターをまとめて解除）
・基本モンスター以外にフィルター（触手宮に初期から設定されていたモンスター以外にまとめてフィルター）

●その他
・エネミー追加方法にフィルターについての記述を追加

ついでに、デバッグ中にウォッチャーの（眼球）が表示されないはずなのに表示されるバグを見つけたのでログを同梱しています。
