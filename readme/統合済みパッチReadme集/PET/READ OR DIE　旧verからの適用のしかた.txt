Ver1.08以前からVer1.2に乗り換える人用、更新に伴っての作業
必ず！！！！！！！　やらずにバグ起きても知らんぞ！！！！！！！！！！

手順
1.まとめ・修正ファイル・パッチなどを全て適用して最新の状態にする
2.パッチによる追加フォルダ内にある「PET」というフォルダを一度削除する（もう使ってないです）
3.BATTLEフォルダ内にSHOW_HEAL_ITEM_LISTがあったら消す
4.ERBフォルダとCSVフォルダを適用
5.既存のセーブデータでプレイする場合は、ペットと別れる（互換性がないので）
6.おめでとう！　手順完了です！　手間かけさせてごめんね！
