触手ソムリエパチュリーさんによるペット名鑑
そのうちエネミー図鑑みたいにゲーム内に組み込みたいけど遥か遠くの話

見方
・名前
　どのように敵を狙うか
　行動率・命中率・レイプ率の高さと
　レイプ攻撃に関しての特筆事項（あれば）
　攻撃に関しての特筆事項（あれば）
　固有スキルについて



・リビングバイン、スカーレットバイン
　ランダムに敵を狙う。
　行動率・命中率・レイプ率普通。
　大振り傾向。低ランクでひるみ、中高ランクで全体攻撃。
　固有スキル「触手の怒涛」は速度が技威力と命中率に上乗せされるが、リビングバイン系は元が速くないのであまり威力は出ない。

・ワーム、サンドワーム
　弱っている敵を狙う。
　行動率高、命中率低、レイプ率普通。
　大振り傾向。全ランクでひるみ攻撃。
　固有スキル「丸かじり」は威力も命中も高い単体攻撃。特に効果も無く全体技でもないので地味だが、強い。
　サンドワームのみ固有スキル「目潰し」あり。当たれば必ずひるむ（相手が耐性もちの時を除く）。

・オクトパス、クラーケン
　戦闘素質の高い敵を狙う。
　行動率普通、命中率高、レイプ率普通。
　命中傾向。
　固有スキル「タイダルウェイブ」は命中率の高い全体攻撃。
　固有スキル「吸収」はダメージを与えつつ、ランクに応じて味方のHPを回復する。高ランクではMP回復に変化。

・イソツビ、アクティニア
　戦闘素質の低い敵を狙う。
　行動率低、命中率普通、レイプ率低。
　バランス傾向。
　全ランクで固有スキル「タイダルウェイブ」を使用する。高ランクになると使用率がかなり高くなる。

・スライム、ブロブ、レッドプリン、ショゴスセル
　弱っている敵を狙う。
　行動率普通、命中率普通、レイプ率高。ブロブは極高。
　命中傾向。高ランクで全体攻撃。
　固有スキル「侵食する軟体」は中ランク以降で使用。低威力だが対象の戦闘素質を恒久的に引き下げる。

・アルラウネ、ドリアード・ローズフェアリー
　ランダムで敵を狙うか、ランダムで味方を回復。
　行動率微低・命中率微低・レイプ率普通。
　ローズフェアリーはレイプ時に衣服攻撃か寄生攻撃しか使用しない。
　命中傾向＋回復スキル。低中ランクでHPとTPを回復。高ランクだとMP回復追加。ただし攻撃系の技は低ランクの弱・中しか使用しない。
　固有スキル「誘惑の芳香」は中ランク以降で使用。ダメージ0、全体にひるみ。
　固有スキル「癒しの胞子」は高ランクで使用。全体のHPを回復する。
　現状、回復スキルを持つのはこの三体のみ。物珍しくはあるが、ほしい時に発動しない回復に何の意味があるのか……

・シェイプシフター、ドッペルゲンガー、ビオラキメラ
　ランダムに敵を狙う。
　行動率極高・命中率普通・レイプ率極高。
　シェイプシフター、ドッペルゲンガーはレイプ時の複合攻撃、寄生攻撃の使用率が高い。ビオラキメラは汚染攻撃使用率が高い。
　バランス傾向。低ランクから高ランクの二段攻撃を多用する。高ランクではひるみも織り交ぜる。
　固有スキル「異形の一撃」は威力・命中率が敵能力に依存し、補正などが掛からない。

・サンドバタフライ、スモークミスト、レディポイズン、アグー、パラサイトミスト
　速度の高い相手を狙う。
　行動率普通、命中率微高、レイプ率低。アグーは行動率微高、命中率低、レイプ率高。
　スモークミストはレイプ時の寄生攻撃使用率が高い。
　回避・速度低下攻撃を好む。汎用攻撃のランクアップが一段階遅い。
　四者とも中ランク以降で固有スキル「目潰し」を使用する。

・マンイーター、フェアリーイーター
　戦闘素質の高い敵を狙う。
　行動率低、命中率普通、レイプ率低。
　フェアリーイーターはレイプ時の汚染攻撃使用率が高い。
　バランス傾向。中ランク以上でひるみ攻撃。
　全ランクで固有スキル「丸かじり」を使用するが、確率は低め。

・ダークアイ
　戦闘素質の高い敵を狙う。
　行動率低、命中率極高、レイプ率高。
　大振り傾向。全体攻撃を好んで行う。
　固有スキル「鑑賞会」は敵全体にひるみ、味方に発情を付与する。

・ミステリードール
　ランダムに敵を狙う。
　行動率普通、命中率微高、レイプ率普通。
　バランス傾向。全ランクで二段攻撃を使用する。
　固有スキル「ふしぎなおどり」は中ランク以降で使用。敵全体の戦闘素質を恒久的に下げるが、味方のTPを下げる。

・テンタクルス
　ランダムに敵を狙う。
　行動率、命中率、レイプ率高。
　大振り専門。全ランクで全体攻撃と二段攻撃を使用する。
　中ランク以降は固有スキル「触手の怒涛」「タイダルウェイブ」を使用する。バイン系よりさらに遅いため、触手の怒涛は威力が出ないが……。

・ダブルイール
　元気な敵を狙う。
　行動率、命中率、レイプ率低。
　大振り傾向。中高ランクで全体攻撃を使用する。
　固有スキル「押しつぶし」は高威力低命中の全体攻撃。

・エンジェルドーター
　ランダムに敵を狙う。
　行動率普通、命中率微高、レイプ率低。
　バランス傾向。全ランクで二段攻撃を使用する。
　固有スキル「戦のラッパ」は中高ランクで使用。アイテム「興奮剤」と同じ効果をもたらす。

・ミミック、人食い箱
　遅い敵を狙う。
　行動率低、命中率高、レイプ率高。
　レイプ時の寄生攻撃使用率が高い。人喰い箱は顕著。
　大振り専門。全ランクでひるみ攻撃を使用する。
　固有スキルは「丸かじり」。

・アルケニー、ワイヤーゴースト
　速い敵を狙う。
　行動率、命中率、レイプ率普通。
　アルケニーは中ランクエネミーだが、行動率・命中率・レイプ率は低ランクなみに設定されている。
　ワイヤーゴーストはレイプ時に寄生攻撃を使用しない。
　速度低下攻撃のみ使用する。
　固有スキル「目潰し」を中ランク以降で使用する。
　固有スキル「必殺仕事人」は高ランクで使用する。超高威力超低命中の単体攻撃。つのドリルみたいなものである。

・スノウガール
　速い敵を狙う。
　行動率、命中率、レイプ率普通。
　速度低下攻撃のみ使用する。
　固有スキル「コールドスリープ」は中ランク以降で使用する。敵の速度を恒久的に下げる攻撃。

・サンダーラプター
　戦闘素質の高い敵を狙う
　行動率、命中率、レイプ率極高。
　レイプ時の寄生攻撃使用率が非常に高い。
　命中傾向。ひるみ攻撃を全ランク、全体攻撃を低・中ランクで使用。
　固有（現状、専用）スキル「マハジオダイン」は高ランクで全体攻撃の代わりに使用する。高威力でひるみ付きの全体攻撃。

・バンシー
　ランダムに敵を狙う。
　行動率普通、命中率微高、レイプ率低。
　レイプ時に寄生攻撃を使用しない。
　バランス傾向。全ランクで二段攻撃を使用する。
　固有スキル「恐怖の叫び」は全体ひるみ付与で、成功率は高いが味方も対象になる。使用率はやや低め。

・サキュバス
　ランダムで敵を狙う。
　行動率、命中率、レイプ率極高。
　レイプ時に寄生攻撃を使用しない。
　バランス傾向。全ランクで二段攻撃を使用する。
　固有スキルは「誘惑の芳香」。使用率は低い。

・ナイトメアケージ
　弱っている敵を狙う。
　行動率、命中率、レイプ率低。
　命中専門。全ランクで全体攻撃。
　中高ランクで固有スキル「丸かじり」を使用。

・レッサーヒュプノ、ウォッチャー、ヒュプノ
　戦闘素質の高い敵を狙う。
　行動率、命中率普通、レイプ率高。ヒュプノはレイプ率最高。
　ウォッチャーのレイプ時行動はレッサーヒュプノと同一。
　全体攻撃のみ使用する。
　固有スキル「鑑賞会」を使用。
　レッサーヒュプノとヒュプノは固有スキル「奉仕催眠」を使用。味方を催眠状態にする技。実質、こちらへの攻撃。
　ペットというより、獅子身中の虫である。

・コウノトリ、フコウノトリ
　ランダムで敵を狙う。
　行動率高、命中率低、レイプ率極高。フコウノトリは命中率極低。
　レイプ時に寄生攻撃のみ使用する。
　コウノトリはバランス傾向。フコウノトリは大振り専門。ともに通常攻撃のみ使用する。
　コウノトリは固有スキル「スペルマシンガン」を全ランクで使用。単体5連撃。
　フコウノトリは固有スキル「スメグ魔砲」を使用。単体高威力。

・ユニコーン、スキュラ、ナインテイルズ、ハウンド、ラミア
　元気な敵を狙う。
　行動率普通、命中率高、レイプ率高。
　バランス傾向。二段攻撃を使用する。
　ユニコーン、ハウンドは固有スキル「獣の連撃」を使用する。単体三連攻撃。
　ナインテイルズは固有スキル「狐火乱舞」を使用する。全体二連攻撃。
　スキュラ、ラミアは固有スキル「触手の怒涛」を使用する。

・ドリフター
　ランダムに敵を狙う。
　行動率、命中率、レイプ率極低。
　レイプ時に性攻撃・寄生攻撃を行わない。
　戦闘中になにもしない。地団駄をふもうが土下座をしようが何もしない。コイキング状態である。
　固有スキルは無い。
　経験値稼いでアイテム生産する人用。

・男傀儡
　ランダムに敵を狙う。
　行動率極高、命中率低、レイプ率極高。
　レイプ時の寄生攻撃使用率が非常に高い。
　通常攻撃のみ使用する。大振り傾向。
　固有スキル「異形の一撃」を使用するが、確率は低い。

・ジェリーフィッシュ、アメーバ
　弱っている敵を狙う。
　行動率低、命中率低、レイプ率普通。
　通常攻撃のみ使用する。
　中ランク以降で固有スキル「侵食する軟体」「吸収」を使用する。

・フレッシュゴーレム
　ランダムに敵を狙う。
　行動率高、命中率極低、レイプ率高。
　通常攻撃は極めて大振り傾向。全体攻撃を好む。
　固有スキル「押しつぶし」を使用する。

・Ｃワンプ、Ｃアナコンダ
　遅い敵を狙う。
　行動率、命中率、レイプ率普通。
　レイプ時に寄生攻撃を使用しない。
　ひるみ・回避低下攻撃を行う。通常攻撃は命中を優先する傾向。
　固有スキル「獣の連撃」を使用する。爬虫類は獣とは違う気もするけど。
　蛇だし毒ダメージとか再現出来ればよかったけど変更箇所がめっちゃ増えるので見送り。

・ホフゴブリン
　弱っている敵を狙う。
　行動率、命中率低、レイプ率極高。
　レイプ時の寄生攻撃の使用率が非常に高い。
　命中傾向。全体攻撃をたまに行う。
　固有スキル「集団の脅威」は中高ランク以降で使用する。超低威力の全体5連。廃レベルまで育てると強い……？

・レイニーバイン、スコールバイン
　ランダムに敵を狙う。
　行動率、命中率、レイプ率高。
　レイプ時の汚染攻撃使用率が非常に高い。特にスコールバイン。
　全ランクで全体攻撃と二段攻撃を使用する。
　固有スキル「触手の怒涛」「スペルマシンガン」を使用する。触手の怒涛はやっぱり威力が出ない……。

・ヴォジャノーイ、ギガヴォジャノーイ
　戦闘素質の低い敵を狙う。
　ヴォジャノーイは行動率高、命中率、レイプ率普通。ギガヴォジャノーイは行動率極高（ヴォジャノーイと等しい）、命中率普通、レイプ率極高。
　全体攻撃のみ使用する。
　固有スキルは「タイダルウェイブ」「押しつぶし」。ランクが上がっても使用率は伸びないが、最初から高め。

・セイレーン
　敵に対して攻撃しない（内部的には速い敵を狙う）。
　行動率低、命中率普通、レイプ率低。
　固有スキル「清浄なる唄」のみ使用する。部屋の汚染度を下げる。
