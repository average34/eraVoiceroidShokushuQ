基本編

・触手をペットにするメリット
　戦闘に参加して、敵への攻撃や味方の回復などを行なってくれます。
　Ver1.2からは敵にあった「ランク」の概念をペットにも導入し、さらに各ペットに応じた固定ステータスや思考、スキルの選択が行われるようになりました。今後も拡張予定です。
　他にも成長要素やアイテム生産など、触手宮のプレイの幅が一気に広まります。

・ペットを捕まえるには
　ひたすら戦って倒してると、たまに起き上がってこっちを見てきます。その時勧誘してください。……要はドラクエ形式です。
　ランクの高い敵ほど捕まえにくくなっていますが、ヒュプノとドリフターは必ず起き上がります。
　なお、「クイーン、ガーディアン、バイオプラント、娘」の四種はペットにできないのであしからず。

・ペットの才能
　ペットを仲間にしたとき、その子の才能に応じた評価メッセージがくだされます。
　これは初期ステータスにどんな風にボーナスorペナルティが出たかを示すものです。

・ペットのステータス
　初期ステータスはペットの種族の能力に応じて決まります。
　それにプラスで、ペットの才能次第で幾らかのボーナスが振り分けられたり、逆にペナルティが入ったりします。

・ステータスの意味
　ランク		ペットのランクです。ペットの初期ランクは、その才能に応じ、本来の種族のランクから若干前後します。
　レベル		ペットのレベルです。
　戦闘・速度	幻想少女たちのそれと大差ありませんが、速度は命中率に補正します。命中させたければ速度を上げましょう。
　やる気		戦闘中の行動確率に気休め程度の補正をします。
　カッコ良さ	クリティカルの確率に関わります。クリティカルが出ると必中で三倍ダメージです。
　満腹度		名前の通りです。アイテムを餌として与えると増え、時間経過で減ります。
　栄養度		名前の通りです。いわゆる経験値で、いろんなことに使えます。
　基礎行動率	マスクデータです。ペットが戦闘中に行動する基礎的な確率で、種ごとに固定で、変動しません。
　基礎命中率	マスクデータです。ペットの攻撃の基礎的な命中率で、種ごとに固定で、変動しません
　基礎レイプ率	マスクデータです。ペットが幻想少女に襲い掛かってくる確率で、種ごとに固定で、変動しません。
　好感度		マスクデータです。色々なことに補正をかけます。ペットが襲い掛かってくると増えて、戦闘中に行動することで減ります。

・ペットと戦う
　ペットは自動でついてきて、自動で戦ってくれます。
　ただし、何もしてくれないことも結構あります。そのへんは基礎行動率やランク、満腹度その他もろもろの影響を受けています。

・スキル、固有スキル
　ペットは種類によって使用するスキルが違います。さらに、種族固有の「固有スキル」を使うこともあります。
　たとえばサンダーラプターの固有スキルは「マハジオダイン」、ドッペルゲンガーなら「異形の一撃」です。
　現状では固有スキルが被ってる種族もありますが、そのうち差別化を図っていきます。
　詳しいスキルについてはペット名鑑に書いてあるから、チェックしてみてはどうでしょう。

・餌やり
　ペットだって生きているので、お腹が空きます。なので、適度にアイテムを餌として与える必要があります。あまりにも放置してるとレイプされます。
　餌がどうしても手に入らない場合は、幻想少女たちの身体を餌として与えることもできます。性的な意味で。
　基本的に、レアなアイテムほどお腹を満たしますが、ペットの餌用アイテム「アオカン」は特別にお腹を満たしてくれます。ネコカンみたいなもんです。
　他にも、種系アイテムを与えるとステータスが伸びたりします。幻想少女たちが食べた時に比べると微妙ですが。

・栄養
　ペットは倒した敵や餌、幻想少女たちの喘ぎ声を栄養として取り込みます。
　栄養が一定以上あると、ランクやレベルを上げたり、アイテムを生産することができます。

・レベルアップ
　栄養を消費して、ペットのレベルを上げられます。
　レベルが上がると、好きな能力値を1上げられます。幻想少女たちのようなステータスになるのは遠い道のりですが……。
　なお、レベルが高いほど、得られる経験値は低くなります。レベルアップは計画的に。

・ランクアップ
　栄養を消費して、ペットのランクを上げます。
　ランクがあがると、5ポイントの能力値ボーナスを任意で振り分けられます。
　ただし、ランクが高くなるほど、ランクアップに大量の栄養が必要になります。

・アイテム生産
　栄養を消費して、アイテムを生産します。といっても、なんでも生産できるわけじゃなく、レートは厳しくなっています。
　それでも欲しいアイテムを確実に手に入れられるのは便利な話でしょう。。
　また、いくつかのサービスも受けられます。武器を手に入れたり、目を覚ましたりできるそうです。


豆知識編

・ランクを上げるメリット
　ランクを上げるメリットは能力値ボーナスだけではありません。実は、ランクが高いほど、戦闘時に色々なプラスの補正がかかるようになっています。
　例えばランクが高いほど、戦闘中に使用するスキルが優秀になり、命中率も補正されます。さらにいうと、行動する確率や命中率は、ランクが低いうちは上限がかけられています。
　ランクを上げれば、能力値以外の様々な麺でも強くなれるといえるでしょう。

・ランクを上げるデメリット
　では、ただランクを上げればいいのかというとまた話が変わってきます。
　ランクを上げると確かに強くなりますが、様々なデメリットもあります。
　たとえば戦闘で得られる栄養が渋くなり、空腹になるのが早まり、襲ってくる確率が高まったりします。
　ランクアップは計画的に。
　ちなみに、内部的には、ランク123が低ランク、45が中ランク、67が高ランクとして区分され、この区分で色々なことが判定されていたりします。参考にしてみてはいかがでしょうか。

・最初から高ランクのペット
　高ランクの敵ほど、基礎行動率と基礎命中率が低めに、基礎レイプ率が高めに設定されています（振れ幅はありますが）。
　上記のデメリットの件もあるので、扱いやすいとはいえなくなっています。
　さらにいうと、高ランクの敵は才能が決定される際、ランクが下り気味になるよう設定されています。
　苦労してランク7の敵をペットにしたのに、いざステータスを見てみるとランク2……ということもあるわけです。

・新規に敵を追加した場合
　パッチなどで新規に敵を追加した場合、そのままだとその子はペット時にリビングバインの行動しかとりません。
　資料などを参考に、必要なファイルを作成することで、ペットらしい動きをするようになります。
