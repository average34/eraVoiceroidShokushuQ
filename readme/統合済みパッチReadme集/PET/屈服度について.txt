屈服度について
　下限0、上限100、具体的な数値はマスクされており、ショップメニューの「ペット」の項目からぼんやり確認できます。
　ペットに犯されることで増え、時間経過で減少します。
　高いほど以下の補正が発生します。

　・行動率、攻撃力、命中率増加
　・襲ってくる率増加
　・襲ってきたときの攻撃力増加
　・レベルアップとランクアップと各種生産に必要な栄養の増加
　・ペットに中出しされて孕む率増加
　・高すぎるとペットを手放せなくなる

　また、屈服度が90に達すると、そのフェイズ終了時に素質【ペット】を取得します。
　ペットを取得すると屈服度が下限50、上限150になり、一部メッセージが変化します。

　高屈服度ではペットは（補正により）強力になりますが、それ以上にデメリットが大きいため、屈服度に気を配る必要が出てきます。
　そのため、相対的に「身体を捧げる」の価値は下がります。といってもわりとすぐ下がるので、狙って素質取得するなら酒と気付薬が必要になりますが……。
