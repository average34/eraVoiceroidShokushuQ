触手宮101用ペットシステム追加パッチ （製作者　喚く狂人）Ver1.07

・概要
	触手をペットとして飼えるようになります。
	ペットは戦闘に参加し、さまざまな行動でプレイヤーを助けてくれます。
	また、ペット用のアイテム「アオカン」を追加します。猫缶みたいなもんです。
	今後は技方面の充実（ペット専用技・ペットの種族に応じた技の追加など）を考えています。バグが無ければ。
	詳しい内容は説明.txtを見てください。

・適用の仕方
	そのまま放り込んで上書きしてください。

・注意
	ERHファイル使用（広域変数定義のため）
	DA:0台を上書き（本来テスト触手が入っているところ。ペットの種族のデータを格納する）
	バランスは……どうだろう。そこそこ渋め、救済措置ありって感じになってるといいんですが。

・まとめとかへの導入など。
	むしろお願いします。
	改変もご自由に。特に今回は凄まじくコードが汚いので、誰か上手く書けるひとがわかりやすくしてくれるとありがたいなーなんて。
