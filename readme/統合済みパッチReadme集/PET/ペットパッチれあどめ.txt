触手宮101用ペットシステム追加パッチ （製作者　喚く狂人）Ver1.3.6

・概要
	触手宮にペットのシステムを追加するパッチです。
	ペットは一緒に戦ったりしてくれますが、餌をあげないとひどい目にあいます。
	他にもペット用のアイテムなどを追加します。

・適用の仕方
	最新環境にﾎﾟｲ

・注意
	ERHファイル使用（広域変数定義のため）
	DA:0台を上書き（本来テスト触手が入っているところ。ペットの種族のデータを格納する）
	CFLAG:400、401を使用
	バランスは僕がこんな感じがｲｲﾅｰって思う感じにしてありますが、ヌルゲーマーゆえヌルいかもしれません

・まとめとかへの導入など。
	むしろお願いします。
	改変もご自由に。特に今回は凄まじくコードが汚いので、誰か上手く書けるひとがわかりやすくしてくれるとありがたいなーなんて。

	ver1.3.6 (別人)

		(タンブルバイン、根無し草、チェイサーバイン、テンタクルウォール、タングルフット、マンドレイク、オオガエル、クロヤマガエル、ホワイトスライム)
		・行動ルーチンはそれぞれ既存のペットから拝借しました。不都合があるようでしたら適宜調整してください。似たようなタイプの居ない敵(ファンガス、プロトクリーナー等)については未実装です。
		・ボス系とペット行動定義ファイルの無い敵以外をペット可に変更。
		・ペット行動定義ファイルが無いがペット可能だった敵は暫定でペット不可に変更。
	ver1.3.5
		・PT内の素質【ペット】取得者数が襲ってくる率に影響するように。

	ver1.3
		・屈服度導入。それに伴う各種補正導入。
		・屈服度が高いとペットを手放せなくなるように。
		・素質【ペット】追加。屈服度による補正が強烈になります。
		・主人公が素質ペットを所持していない場合は全滅時に確率でペットを失うように。
		・ペットに襲われて孕むように。ペットに孕まされると、そのフェイズ終了時に特殊メッセージが表示され、屈服度が大幅上昇。任意のタイミングで妊娠できてしまうため、確率は低め
		・月経パッチが原因で、射精が寄生攻撃に化けてたことがあった？　っぽい？　ので修正（The code works, but I don't know why)
	ver1.21
		・行動率の補正式がちょっと間違ってたので修正。
		・ペット加入を断れないバグを修正。
		・多少のバランス修正。
		・ホフゴブリンのデータを同梱。
	ver1.2
		大型アップデート。ほとんどリビルド。多分そこそこまぁまぁ分かりやすいコード……きっとね。
		以下はプレイしてて分かるもの。
		・ランクのシステムが追加された。いろんなものがペットのランクに依存するようになった。
		・ペット加入時の才能評価システムを追加。コメントが出たりするようになった。代わりに最大HPでのボーナスが渋めに。
		・栄養の取得をだいたい上方修正。入りやすくなった。あと、餌やりで経験値が入るように（微量）。
		・「身体を捧げる」の取得経験値が、捧げたキャラの感覚レベルや中毒レベルに依存するように。
		・各ペットに固有の思考とスキルを搭載。敵追加パッチを作成する場合は一手間必要になった（省いてもいいけど）。
		・上記に伴い、作戦システムはオミットされました。。
		・SELECT_ENEMY_ACTIONを読みに行くのをやめたので、レイプ周りの安定性が上がった気がする。前みたいにエラー落ちはしないと思う。多分。
		・喪失OR催眠のキャラと会話中に、ペットに襲わせることが可能に。好感度はガン下がりする。
		・アイテム生産に「腹を満たす」「目を覚ます」が追加。便……利？
		・バグフィックス系パッチは（そのままだと取り込めないので）リビルドの際に参考にさせていただきました。
