■ パッチ内容
1.02.0で追加された手書き地図を拡張するパッチです
通常マップの部屋番号をクリックする事で直接メモを編集できます

また、部屋情報の下に現在地のメモを表示させることが出来ます



■ 変更詳細
SHOP.ERB
	SHOP - 手書き地図を直接編集モードで呼び出せるように判定を追加

PRINT_MAP_INFO.ERB
	通常表示のマップも手書き地図同様に番号クリックが可能に
	現在の部屋の色表示・メモ表示を追加
	
HAND_MAP.ERB
	SHOP - 手書き地図から以外でも呼び出せるようにHAND_MAP_MAIN関数を改変
	引数に部屋番号を渡してやると直接編集が可能に
