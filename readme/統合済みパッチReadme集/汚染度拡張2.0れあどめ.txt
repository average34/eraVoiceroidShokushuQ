汚染度拡張パッチver2.0

・概要
　ホフゴブに続いて汚染度拡張パッチが生まれ変わりました。つってもあっちと違ってリファクタリングですが。
　基本的には、汚染度を気にする必要性が更に高まります。また、低難易度は優しく、高難易度は（多分）さらに難しくなります。


以下、具体的な内容
・各種判定に用いていた基準の分布を変更（超大事！）
　旧版だと0,1~10,11~40,41~60,61~99,100で、10以下ならプラス、41以上でマイナス補正だったのが、
　新版だと0~5,6~20,21~60,61~80,81~99,100で、20以下ならプラス、61以上でマイナス補正になります。
　プラス補正を受けるのが低難易度でも辛く、逆にマイナス補正を喰らいやすすぎたため設けた処置です。

・一部の敵に関して、戦闘中の汚染度増加・回復・ダメージの対象外に。
　具体的にはクイーン（腕）とヒュプノです。
　クイーン（腕）については、クイーン戦の辛さ緩和のため。
　ヒュプノについては、汚染度上昇→高汚染度での敵回復　が凶悪に働き過ぎるためです。
　多少は楽になると思います。……焼け石に水程度ですが。

・汚染度による敵へのダメージ/回復が、若干のランダム要素を受けるように。
　誤差程度ですが若干の差が出ます。

・汚染度による敵へのダメージ/回復は、難易度の影響を受けるように
　低難易度だとダメージは高く、回復量は低くなります。高難易度だと逆です。

・汚染度MAX時の敵の回復の仕様変更
　旧verでの汚染度MAX時の敵の回復の仕様は「敵HPの5%（最低1000）回復」でした。
　汚染度が普通に高い時との違いは、最低値が倍になっているということだけでした。
　新しいバージョンでは、汚染度MAXでも最低値は変動しません。ただし、回復量が倍です。
　これは難易度による修正も倍になって乗るので、H以上では汚染度MAXで戦闘を続けるのは困難になるでしょう。

・汚染度による敵へのダメージ/回復量が表示されるように
　色もついて見やすくなります。たぶんね。

・汚染度による味方の行動のダメージ、敵の行動のダメージ/命中率への補正は、難易度の影響を受けるように
　今まで受けてませんでしたが変更しました。基本、Nが今までの式で、難易度により修正を受けます。
　あと式自体の変更も加わってます。

・汚染度による味方の命中率への補正は式を変更
　Nで若干の難易度減少が起こります。あとこれに関してはもともと難易度の影響を受けてます。
