触手宮1002まとめパッチ用触手服常駐時のメッセージを修正するもの （製作者　喚く狂人）

・概要
　着ている触手服の種類がなんだろうが、常駐時のメッセージで「ボディースーツがどうちゃら」というのが出てきてしまうことがあるのを修正し、ちゃんと着てるものの名前が出てくるようにします。
　服装パッチONじゃない場合はボディースーツと表示されます。

・適用の仕方
　そのまま放り込んで上書きしてください。

・まとめとかへの導入
　してくださると後日僕が「しまった！　まとめ導入した時に上書きした！」とかなるのがなくなってハッピーになれます。
