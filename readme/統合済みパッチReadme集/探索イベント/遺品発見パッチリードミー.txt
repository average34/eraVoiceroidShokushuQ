仲間が完全に捕食された後、思い出しか残らないなんて寂しいじゃないですか；；
そんなわけで遺品が発見できるパッチです

サニー「ねえスター、これはどんなパッチなの？」
スター「例えば前半でうっかりイソツビに遭遇しちゃってどうしても勝てないとき、ルナが丸呑みされちゃったとするでしょう？」
ルナ「なんで私を例にするのよ！」
スター「(ﾆｺﾆｺ)そうしたら仕方ないから私とサニーの犠牲を無駄にしない為にも逃げるわけ」
サニー「うんうん」
ルナ「酷い！」
スター「武器も揃った、アリスさんみたいな強力な助っ人も仲間になった、さあルナを助けにいくわよ！と部屋に戻ったら誰もいない」
サニー「まったく、私達の助けを待たずにどこかいくなんてルナもせっかちね」
ルナ「それってもう食べられちゃったんじゃ……」
スター「ルナはどこに行ったのかしら？『探索』で必死に部屋を探す私達……」
ルナ「だからもう食べられちゃったんじゃ…」
スター「そこでこのパッチの出番よ♪私達は必死の捜索の結果、白濁液にまみれたルナの帽子を見つけることができるようになるのよ！」
ルナ「私自身はみつからないの！？」
スター「見つかるわけないじゃない＾＾遺品だもの、持ち主には一回休みになってもらわなくっちゃ」
サニー「へえ、おもしろそう！さっそくルナを前進させて戦ってくる！ところで、遺品を見つけるメリットってあるの？」
スター「ただの雰囲気作りで何も」
サニー「えー……」
スター「二人の遺志をついで最強の私が触手宮の覇者になれるなんてのも面白いと思ったのだけれども、
		簡単に遺品でパワーアップ！なんてなったらバランス取りが難しそうだし、遺品目当てで仲間を次々と餌にする人がでてきそうだもの。
		それって何か違う気がしてね。探索経験値獲得くらいいれるべきだったかしら……」
サニー「残念ね、こう仲間の死を乗り越えて合体！パワーアップ！覚醒！妖精の地位アップ！なんて王道の熱血展開なのに！」
スター「あとは妖精や蓬莱人の遺品を無限に拾って無限パワーアップできちゃいそうだったのも理由の一つね。
		これは敵がランクアップするし、私達妖精や蓬莱人の遺品のパワーアップ効果を薄くすれば解決できそうだけど。
		サニーが言うみたいに遺志をついでパワーアップ！なんていうのもカッコイイけど、熱血より恐怖、悲しさ、そういうほうが世界観にあいそうじゃない？(ﾆｺﾆｺ)」
サニー「そういうものかしら……」
スター「ふふ、そういうものよ♪」
ルナ(あれ？二人の遺志をついでってスターはパワーアップ機能ついていたら私たちを生贄にする気だったんじゃ……)
