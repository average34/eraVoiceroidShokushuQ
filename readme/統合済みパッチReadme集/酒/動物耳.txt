動物耳パッチ　製作者:まとめの人

動物耳です　ピコピコ動きます

・動物耳が追加されたキャラ
　橙、藍、ミスティア、もみじ、おりんりん、てゐ、優曇華院、レイセン、響子、ナズーリン、星、マミゾウ、朱鷺子

星は動物耳？可愛いからいいんですよ！

・効果
　気配察知と危機回避のレベルに応じて戦闘自体を回避出来ます
睡眠中や戦意喪失中、発情中は無理です
PTに複数動物耳が居た場合全員が効果があります

獣の勘ということでちょっと強めにしています
回避率は気配察知と危機回避が両方0でも1/8
以降それぞれ1レベル増えるごとに分子が+1です
逃げることに関しては強いですが敵が減るわけではないのでその場しのぎになりやすいです
というか橙、ミスティア、もみじ、てゐ、レイセン、響子、ナズーリン、朱鷺子と8/13が戦闘素質40以下なので弱キャラ救済……
戦闘素質高いキャラなら倒してしまったほうがいい場合も多いので弱い獣耳勢が仲間と合流するための素質です


敵が100匹いてもMAX7/8で回避できるのは内緒
治療+調合知識の優曇華院や、哨戒持ちのもみじが便利キャラになっているのも内緒

探索を安全にできるので動物耳3人だと石版エンドは狙いやすいです
眠気やトイレの対策が出来ればですが……


・動物耳入手イベント
すれに出ていたので追加
怪しげな木の実を食べると耳と尻尾が生えます
どの動物のものが生えるかは口上主さんにお任せ（耳と尻尾としか書いていません）

なお動物耳をすでに持っていると発情します
耳が生えた状態で顔を真っ赤にして我慢できずに戦闘中仲間に飛びかかる　いいですね
尻尾の動きで下着が擦れてもじもじする　いいですね

後天的な場合動物耳が2になります
