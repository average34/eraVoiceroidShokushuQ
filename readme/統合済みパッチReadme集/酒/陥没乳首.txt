陥没乳首追加

母乳を吸われない+Bダメージ減少という恵まれた素質です　素質番号159
ただし存分に愛撫された状態でBダメージを受けると乳勃状態となり母乳を吸われる上Bダメージが大幅に増加します
また愛撫が足りなくても一定確率で乳勃状態になります
戦闘中の状態異常の欄で乳勃状態かどうかは確認できます
利用したい場合はコンフィグ設定で陥没乳首を設定してください
全員を選ぶと現在のキャラクター全員　ランダムを選ぶと現在のキャラクターが半々の確率で陥没乳首だと発覚します
グローバルセーブに記録させると2周目以降でも同じ設定で陥没乳首になります

母乳が吸いにくいので陥没乳首だと出産時のB絶頂が2回になります

色々迷ったんですが陥没乳首は先天性のようなのでコンフィグという形に
自販機と出産での陥没乳首付与はしてもいいかもしれません
