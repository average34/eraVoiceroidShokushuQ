おしりに寄生パッチ　製作者:まとめの人

たまごや種子をお腹に植えつけて寄生→産卵が起きるパッチです
口やお尻から入れて一定期間後にお尻から出てきます
コンフィグにより通常の寄生と同じメッセージになります

寄生との違い
・服が破けない
・胸が大きくならない
・母乳体質にならない
・妊娠や寄生と同時に発生
・メッセージが別

お尻への寄生攻撃を作る場合はほとんど通常の寄生攻撃と同じです（適当な寄生攻撃をコピーペーストでOK）
ただし
TFLAG:63 = 0
にしてください。
TFLAG:63はメインの性属性で寄生攻撃は通常3のV属性です
0にすることでA属性に変更されます

使用変数

FLAG
533	産卵プレイONOFF

CFLAG
151	抱卵(A寄生)　数値で父親の種類
152	A寄生タイムリミット　寄生や妊娠と重複するので別管理
