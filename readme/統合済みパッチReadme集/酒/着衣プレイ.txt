着衣プレイパッチ　製作者:まとめの人

触手さんが損傷～中破くらいで服を破くのをやめるモードをコンフィグに追加します

・やり方
パッチのコンフィグにて着衣プレイモードをONOFFできます

・システム面
ONにした場合CPが40%以下だとCPが減らなくなります
ただし全裸の場合の敵の攻撃力増加ペナルティはCP40%以下になった時点で受けます
CP回復系の効果とは相性がいいですがダメージを受けやすくなることには注意してください



・イメージ面
触手さんが着衣プレイ大好きになります
服が残った状態でプレイに勤しむ触手さん

服を破くのをやめた時点でダメージは全裸と同じになりますが
ボタンなどを引きちぎって素肌に攻撃しやすくしたとか、服をずらして敏感な部分を露出させたとか、そういうことです
