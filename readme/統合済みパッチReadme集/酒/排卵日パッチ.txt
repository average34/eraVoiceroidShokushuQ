排卵日パッチ　　制作者：まとめの人


話題として出ていたので作ってみた

月経日を定義して危険日や排卵周期を処理できるパッチです

CFLAG:150が月経日を示す日数です
30n + CFLAG:150 が月経日になります
例えばCFLAG:150が11なら月経日は11日・41日・71日……　となります
血とか出る日ですがこの日自体は妊娠しやすいわけではありません

性周期を30日として排卵が起きるのは16日めあたり
危険なのはその前4日と後1日の12～17日でこのタイミングだと今までと同じかそれより高い確率で妊娠します
（CFLAG:150が11なら23～28日目は襲われるとかなり危険です）
逆にそれ以外の日だと妊娠確率はこれまでより下がります

難易度下がるかというとそうでもなく
寄生が起きやすくなるので薬酒が十分な量手に入らず治療が間に合わなかったり、
妊娠すると拘束から開放してくれるコウノトリが処理の都合上開放しにくくなったりしています
