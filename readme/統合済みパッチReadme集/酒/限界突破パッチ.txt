限界突破パッチ　　製作者　触手宮酒パッチの人

ネタです　バランス次第でまとめに組み込みます
敵から受ける快感ダメージ、キャラのMP、敵の攻撃力が限界突破するようになります。

敵から受けるダメージ
各部位の感覚が3になっても経験値に比例してダメージが増えていきます
正確には　経験値の根号/100　だいたい感覚レベル3直後で2.2倍、経験値500000(レベル3に必要な量の10倍)で7倍ほど
MPは割合ダメージが乗るようになります

キャラのMP
淫乱時の回復でオーバーした分の20％だけプラスになります

敵の攻撃力
母乳でHPが限界突破し120％を超えたら、HPが最大値に戻り戦闘素質が120％になります
