M感覚と肌感覚追加パッチ　製作者:まとめの人

口とか脚とか肩とか耳とか首筋とか鎖骨とか背中とか攻める攻撃を追加できるパッチです
実際の攻撃追加の前に処理だけ前倒し
これでディープキスとか全身舐めまわす性攻撃ができます
攻撃のバリエーションが増えるだけじゃなくダメージの一極化が起こりにくいので長時間滞在が少しだけしやすくなるかもしれません



追加の仕方
TFLAG:63	メイン性属性　で
5を指定すると肌（脚とか肩とか耳とか首筋とか鎖骨とか背中とか要はどこでも）
9を指定するとＭ（他バリアントにある口感覚）
に攻撃できます

数字が飛んでいますが少しでも処理を楽にするための仕様です
肌属性はダメージが触手中毒依存　攻撃を受けた際も触手中毒が増え、少しだけEPダメージが増えにくいです

サブ性属性で指定する時は
TFLAG:92	のビット演算で
16が肌
32がＭ
になります

またTalentの
79,キス魔	Mに対応する淫乱素質
105,肌感覚	肌に対する感度
106,Ｍ感覚	Ｍに対する感度
も処理だけ追加したのでcsvをいじることで使うことができます
