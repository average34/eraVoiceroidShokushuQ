-縮乳薬について

触手に寄生されて何度も擬似出産し、私の胸はおかしな大きさになっていた。
少しでも日常に近づけたかったためにこの薬に手を出す。
しかし触手の攻めは止むわけでもなく、何度も何度も苗床にされかける。
そのたびに薬に手を出し・・・でも何の解決にもならない。
むしろ薬の反動のせいかどんどんと膨らみ方がひどくなって・・・

という感じがやりたかったんです。

-魔法の拘束具について
本来敵に使用しますが触手相手だと拘束から抜けられてしまいます。
しかし電撃を弱めに設定することでNPCの回復に使うことができます。(いつまでも触手に犯され続けます)
主人公たち以外には外せないので敵に連れ去られることがありません。(不死身モードと違い復活して逃げられることがありません)
これでNPCが全員生き延びた状態でのクリアが用意になります。(性欲発散部屋を触手に邪魔されずに作ることができます)
なお会話コマンドが特殊なものになります。(意識があっても性欲発散などが行えます)

-ヒュプノ
催眠状態を操る敵です
全体催眠後性攻撃や寄生攻撃、拘束攻撃を行います
耐久が非常に高いですが催眠の負荷でそのうち自爆します

ガーディアンの亜種で中心は手のひら大の目玉から大量の触手が生え、大きさはガーディアンよりかなり小さい
能力の反動があるので普段はくらいところに隠れている
小さい分触手の動きが素早いので敵を無抵抗にする特性と相まって複数回攻撃が可能
、ぐらいまで考えてました。
でも触手の先端に目も有りだと思うのでついさっき尻尾状のひときわ太い触手が脳内で生えました。
多分強い性攻撃に使うんでしょう(9/5)

●ヒュプノ
催眠状態を操るガーディアンの亜種の敵です

蛇のように、長い触手の胴体を持ち、上下の顎が大量の触手でできています
舌に当たる部分には人間の体ほどの眼球があり、魔力を込めた視線と媚薬のような体内物質で対象を催眠状態に陥れます
蛇に睨まれた蛙が動けないように、ヒュプノに睨まれるとメスとして苗床として自ら身体を差し出してしまいます

コンフィグで差し出し方を選べますが個体差ということで……

ただ能力が強力な分反動もひどいので普段は暗くて狭いところに縮こまって隠れています

催眠をかけてじっくりと苗床にします

地雷モンスター、能力値だけならトップクラス
でも会うのが大変なのできつすぎるという意見も少なめ
というか対策すればけっこう楽だったり……
