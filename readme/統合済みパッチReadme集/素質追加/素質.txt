話題になった粘液の書を見たら素質絡みでミスがあることに気づいたので修正し、ついでに所持者が少ない素質を強化したもの……
というかいらない子と化している離脱を強化したかっただけで抑圧はおまけ（個人的に抵抗と抑圧は対になってるイメージがあるので）。

主な修正点
・九十九姉妹のcsvに処女があるのを修正（触手宮ではキャラcsvに処女は配置しない、はず）
・辞書の結界知識の所持者を記述
・辞書のアイテムの説明を見る時一旦止まるように（見難い）
・辞書のアイテム一覧の空白を揃える
・辞書のアイテム一覧の改行タイミングがアイテムを選ぶことにズレるのを修正

主な改変点
・[抵抗]持ちは絶頂している状態でも本来の成功率の50%の確率で離脱と力溜めが成功するように
・[抑圧]持ちは絶頂ダメージが1割減るように（淫乱＋対応部位の淫乱素質時除く）

疑問点
・イージーで絶頂ダメージが半減するのにベリーイージーで半減しないのはバグくさいけど仕様だったら困るので手を出せない
