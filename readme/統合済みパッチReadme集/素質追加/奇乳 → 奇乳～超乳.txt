■■ 東方触手宮 奇乳細分化(超乳化)パッチ

触手宮なら奇乳の上があっても良いじゃない
→素質[奇乳]を細分化して超乳まで対応させました



■具体的な変更点
豊乳化イベントがオンになっている時に、TALENT:158の値が1以上になる様にしました(最大8まで)。
既存のセーブデータに影響を与える事はありません。
奇乳細分化にあわせて、TALENT:158が条件の分岐で1のみの指定になっていた部分を省略しました。

寄生・妊娠による出産、粘装、母乳を溜める方法で奇乳レベルが加算されていきます。
(粘装の乳腺浸入による奇乳化は1まで)

1	:奇乳
2～4	:奇乳 [+～+++]
5	:超乳
6～8	:超乳 [+～+++]

奇乳値に比例した値(数値×3)が
ターン終了時の母乳値(TFLAG:506)の加算にプラス補正されます。

縮乳薬の使用による変化について、
奇乳値が5以上の時に使うと-4され、4以下の時に使えば爆乳になります。
母乳値(CFLAG:506)が40以上ある時は、サイズは変わらず母乳値だけが減少します。



素質の説明はとりあえず以下の通りにしてあります。
超乳の説明はeratoho総合スレから採用させて頂きました。

1～2	～[奇乳+]	母乳が溜まりすぎて人の限界をとうに超えてしまった醜い乳房です
3～4	～[奇乳+++]	両手で抱え切れないほど醜く肥大した乳房。母乳をたっぷり蓄えています。おっぱいオバケ
5～6	～[超乳+]	ミルクタンクと化した双乳からは間欠泉の如く白い蜜が迸り続けるでしょう。
7～	～[超乳+++]	乳タンクから滾々と湧き出る母乳は触手へと捧げられ、新たな苗床を育てる事でしょう。
