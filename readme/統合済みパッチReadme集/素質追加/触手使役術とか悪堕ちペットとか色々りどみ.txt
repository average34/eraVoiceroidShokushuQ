東方触手宮Ver1.04.6用の悪堕ち・ペットの強化パッチです。
当然ながらeraT0006939.zipのラミア化追加パッチの導入を前提にしているので注意。
触手宮敵追加パッチ試作版（eraT0006966.7z）とはファイルが競合しているため、
導入の際に「触手宮敵追加パッチ試作版（eraT0006966.7z）を導入済の場合」を参照・使用すること。

主な更新点は通常悪堕ちキャラの復活システム、転生悪堕ちキャラのペット化、新素質・触手使役術の導入。
前二つは文字通り、触手使役術についてはペット加入率・好感度・屈服度などに補正が掛かるペット関連専用の素質となります。
特に加入率はパーティに一人持っているキャラがいるだけでもほぼ倍になりますが、代わりに持っているキャラが希少（六人だけ）なので注意。

また、テストプレイ中にいきなり◯◯の娘が出るバグが出たのでログ同梱。
多分シェイプシフターだと思うけど一応シェイプシフターが娘に化けるのはバグ……って事でいいのかしらん。

更新点兼自分用メモ
◯CSV
・Talentに悪堕ちと触手使役術を搭載
・パチュリー、紫、諏訪子、さとり、ぬえ、せーがに新素質「触手使役術」追加。
　メリーとマミゾウも考えたけど除外。モンスターテイマー的素質作ったし諏訪子から両刀削除。
◯ERB
・ラミア化を追加した際にPREBATTLEに悪堕ちキャラと出会った時の口上関連処理があったのを素で忘れていたので修正
・エンドフェイズの悪堕ち離脱時に悪堕ち素質の習得を追加。
　まだ素質は追加しただけ、今後判別する必要が出る場合に備えて。それに口上で判別したい場合もあるかも
・DELETE_ENEMYで通常悪堕ち系の復活処理を追加、転生悪堕ちキャラをどこかへ消えるメッセージから除外
・ENEMY_103～105でペット化が可能なように修正
・SHOW_TALENT_WIKIに悪堕ちと触手使役術を追加
・SHOW_TALENT_LISTに悪堕ちと触手使役術を追加（前者は見る機会まだないけど……）
・PET_INVITEに転生悪堕ちキャラのペット化処理を追加、触手使役術持ちキャラの加入率補正を追加
・PET_INVITE2に転生悪堕ちキャラ以外がペット化した場合PET:9を漂白する処理を追加
・PET_RAPEに触手使役術持ちキャラの屈服度補正と好感度加算を追加
・アルラウネ化キャラ、スライム化キャラ、ラミア化キャラのペットデータを作成
◯資料とか
・ペットパッチ資料ボックスにPET:9を追加
