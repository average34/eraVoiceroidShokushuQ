これは味方キャラが特定状況下で敵に回るパッチです。
特定の敵によってリタイアした場合、バイオプラントが発生せず代わりにそのキャラが敵化する（処理的には同じ名前の敵が発生する）ようにしてあります。
敵のデータとしてはドッペルゲンガーと娘を合わせて作成。DA46には親のIDではなく味方だった頃のIDを入れるように。
通常の悪堕ちキャラはふたなり化イベントをONにしていないと攻撃のレパートリーが一気に減るのでそこは注意して下さい。

新素質として淫魔を活用及び狂気を導入。更に悪堕ちポイントとしてCFLAG:160を使用。悪堕ちのオンオフとしてFLAG:538を使用。
性経験の上昇や淫乱系素質の習得などで淫魔を習得すると悪堕ちポイントが溜まるようになり、危険な状態になると狂気を習得します。
そして条件を満たすと仲間にいようがいまいが強制リタイアで敵に回る。一応、主人公は淫魔を習得しないのでゲームオーバーにはなりませんが。
そういった関係で淫魔イベントでは遺品関係が発生しないよう変更。
敵の種類としては
・名前がそのままの悪堕ちキャラ（触手を使わずに襲ってくるタイプ）
・触手姫（触手を使って襲ってくるタイプ）
・アルラウネ化（名前の通り）
・スライム化（名前の通り）
の４つ。
更にエンディング用のヒストリーを追加した他、娘や悪堕ちキャラがいる敵パーティは遭遇時や勝利時に判別できます。
とりあえずこれで自分のやりたいことは全て搭載しました。
