いずれも……
ARG:0 == 1、ARG:1 == コマンドの番号、%ARGS:0% == 自分の名前、%ARGS:1% == 相手の名前、ARG:2 == 相手、ARG:3 == 状況
――となっています。
これで思う存分ｷﾏｼﾀﾜｰが書けるようになったわけです。ｷﾏｼﾀﾜｰどころじゃありませんが。
