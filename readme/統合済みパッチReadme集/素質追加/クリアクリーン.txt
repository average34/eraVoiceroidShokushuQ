
●澄んだ泉と白濁泉が四隅に行かないように変更
　クイーンやガーディアンと澄んだ泉の位置が被るというどうしようもない事態が発生しうるので、四隅に行かないように。

●新素質「浄化知識」「結界知識」導入
浄化知識は浄化失敗時の暴走イベントが発生しにくくなる（浄化力250以上で回避）。
結界知識はそれに加えて浄化力250以上で結界の展開が可能になり、
更に3人PTなら他の二人が浄化知識か結界知識を保持していれば浄化力200で結界が張れる。
ただしどちらも戦意喪失時、発情時、妊娠・寄生末期は機能しない。
