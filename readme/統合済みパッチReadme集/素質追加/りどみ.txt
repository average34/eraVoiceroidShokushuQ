一部イベントで触手がクリトリスに寄生してふたなり化します。そういう意味ではC寄生パッチ？
CFLAG:42、TALENT:121、ARG:0の451、452、453、454、455、456を使用しています。

改変修正導入再配布は、後述の内容に沿う形ならばご自由にどうぞ。

●ふたなり概要
・一部のイベントで「ふたなり」の状態異常が発生する。この時点では「素質」ではなく「状態異常」。
　状態異常が発生するのは淫蟲Cイベント、白濁泉イベント入浴、C寄生淫蟲イベント。
　CFLAG:42に一定数値が入り、その回数射精（C絶頂）するまで残り続ける。

・ふたなり状態の時にC絶頂すると射精し、HPを消耗する。消耗の度合いはC感度・C感覚・射精中毒による。
　C過敏とC感度・射精中毒がそれぞれ最大だと一気に9600(!)吹っ飛ぶ。
　そこまで行く事はめったにないだろうけど。

・地の文は一応変更済みで、ふたなり状態だと一部の文章が変わる。見落としがあるかもしれないが……

・ARG:0が451で射精時、452で寄生終了時を見ることができる。
　淫蟲CイベントはARG:2 == 3でふたなりになった瞬間の口上が出せる。白濁泉は6。
　C寄生淫蟲イベントは新規イベント。ARG:1 == 152で見れる。
　ARG:2が1でふたなりになる、2で既にふたなりだったキャラが吸精される、3で回避。

・射精中毒が3になって射精すると取れなくなる。その時の口上はARG:0 == 454。
　その際はCFLAG:42が0になり、代わりにTALENT:121を習得する。こちらに残り回数はない。

・主人公がふたなりで味方がいないor味方が全員戦意喪失かふたなりだと、発見した戦意喪失キャラを犯すことができる。
　EP・MPが回復して残り射精回数も一気に減らせるが、高確率でシェイプシフター孕むし好感度も100前後下がるので注意。
　ARG:0 == 455で口上表示可能。ARG:1に犯したキャラ、ARG:2に射精した回数。
　言うまでもないが、仲間がふたなりだと輪姦になる。仲間の口上はARG:0 == 456、あとは同じ。
