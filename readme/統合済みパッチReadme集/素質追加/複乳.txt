------------------------------------------------------------------------
■ 東方触手宮 素質[複乳]追加パッチ　
------------------------------------------------------------------------

おっぱいが増えればもっと幸せになれると思って作りました。

乳サイズと母乳体質により複乳化の進行度が早くなります。
複乳の効果はまだ特にありません。フレーバー素質。


→ 発生条件
--------------------------------
・テキスト表示の残虐度表現レベルがHard以上で素質取得可能
・コンフィグで複乳パッチをONにしている
・[淫乳]の素質がある
・侵食によりBにインプラントされている
・侵食値が15以上
--------------------------------

→ 発生フラグ
--------------------------------
・発生条件をすべて満たした状態で、スコールバインの攻撃(FIX_ENEMY_ACTION_766)を受ける
--------------------------------

→ 複乳化進行度増加(発生フラグ後)
--------------------------------
・スコールバインの攻撃(FIX_ENEMY_ACTION_766)
・V(妊娠・寄生)の出産
・B寄生orB抱卵の出産
・触手服、スライム服のB攻撃
・ターン終了時
--------------------------------

→ 素質取得機会
--------------------------------
・スコールバインの攻撃(FIX_ENEMY_ACTION_766)
・ターン終了時
--------------------------------

→ 使用している変数番号
--------------------------------
FLAG:598	複乳パッチ
TALENT:230	素質[複乳]	値：1～3

CFLAG:513	素質成長管理

FIX_ENEMY_ACTION_766.ERB	(性攻撃)	スコールバインB+

SYSTEM_MESSAGE_535(複乳化)
--------------------------------
