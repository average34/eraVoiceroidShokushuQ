東方触手宮Ver1.05.3人柱版修正　用連撃パッチ


目的
・非消耗武器の急所狙いでひるませた敵に、消耗武器や弾幕の追撃を打ち込めるようにしたい
・近接武器と射撃武器の連撃とか厨ニ心くすぐるじゃん？
・二刀流も使えるよ！
　発動条件は【剣術・体術習熟】を持ち、それぞれ該当するカテゴリの武器を装備1、2に装備していること
・見た目のかっこよさ重点
　弱攻撃限定なので、上位武器で揃えない限りそんなに酷い火力馬鹿にはならない…はず
・ついでに☆マーク付きのユニーク武器を三種類追加、連撃機能と合わさって最強に見える
　「☆触手放射器」
　火炎放射器の燃料タンクの代わりにオナフォがついてて、装備者から搾精して再装填する
　つまり装備者が「ふたなり」持ちでないと再装填できない
　イメージ映像（マジキチ注意）
　https://www.youtube.com/watch?x-yt-cl=85114404&v=SfFMAtmWb5s&x-yt-ts=1422579428

　「☆触手使役術」「☆触手剣」
　触手服の一部を武器として使役する
　装備者は問答無用で「触装」状態になる（魔法少女の悪い子版）
　イメージ的にはバイオ４のクラウザーさんとか

　ユニーク武器性能
　　☆触手使役術　…　攻撃力50、装備時に触手服、結界で脱いでも次のターンで触手服　淫乱系素質や触手中毒によってダメージ上昇　　蒸　着
　　☆触手剣　　　…　攻撃力60、装備時に触手服、結界で脱いでも次のターンで触手服　淫乱系素質や触手中毒によってダメージ上昇　　蒸　着
　　☆触手放射器　…　攻撃力100、弾数30、リロード時に装備者のチンポミルクを装填　　淫乱系素質や触手中毒によってダメージ上昇　　んっほぉぉぉぉっっ！

   ※触手系武器の性能に影響を与える要素
　　・触手中毒Lv
　　・淫乱系素質全種
　　・乳腺開発＆乳腺拡張
　　・異常排卵巣＆産卵体質（又は卵生）
　　・侵食度＆インプラント
　　・悪堕ち進行度
　　・妊娠＆寄生＆乳卵＆Ａ卵
　　・触手服＆触手包帯＆スライム服
　　・部屋の汚染度
　　・戦闘素質
　　条件が揃えば(相応の手間はかかる)難易度Lunaの重度汚染部屋であっても並みの触手なら一点集中一撃で蒸発する威力になるはず

・ユニーク武器入手条件
　・三種類とも難易度HARD以上前提
　・【☆触手使役術】にょろろに全身開発されてご褒美もらう時、おまけとして入手
　・【☆触手剣】バイオプラント、又はナイトメアケージを倒すと入手（一個だけ）
　・【☆触手放射器】パーティメンバー3人で全員素質【ふたなり※】（ただの状態異常ではダメ）または【男の娘】を持った状態で
　　触手祭壇で「スゴイ☆武器が欲しい」を選ぶと、三人まとめてMP0なるまで搾精される。
　　全員戦意喪失した時点で三人の合計射精量が一定超えたら入手。（確実に全滅するので周囲の安全をよく確認すること）

※各武器の入手条件H以上だが、入手条件が揃った時点でHARDであればいいので、どうしても取れない子猫チャンはコンフィグで難易度変えてもイイのよ？
　・触手放射器はNで全員ふたなり淫乱つけてHで自販機でおうけい
　・触手剣もNでプラント瀕死にした上でHでトドメさせばおうけい
　・ただし触手使役術は、“にょろろが立ち去る時にH以上でないと二度と入手不可能”になるので注意！
　・ユニーク武器は性質上、悪堕ちパッチと併用すると難易度が有頂天になるので用法用量に気をつけよう！

・ガーディアン及びクイーン撃破後にクリアせず、引き返してゲーム続行する選択肢イベントを追加
　本体側で発生する「ガーディアンやクイーンの腕が増えるバグ」の応急処置として個人的に作ったもの
　単純に、ボスをプレイヤーが倒してもエンディング迎えないようにして掃除できるようにしただけ
　選択肢とイベントメッセージ作ったので単純にストーリーとして楽しんでもよし
　増えちゃったボスをお掃除してもよし
　今後バグが修正されたとしても、問題なく動くと思う
