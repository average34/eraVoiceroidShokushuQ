現状、霊撃の関係で非消耗武器使いにとっての戦闘素質が単なるバッドステータスにしかなってないため、
特定の素質を持っていると非消耗武器の戦闘に戦闘素質がある程度乗るようにしたもの。

体術習熟：
弱中強カウンターのダメージ（TFLAG:58のほう）に戦闘素質の20%を上乗せ。
下段狙いの速度低下量に戦闘素質の10%を上乗せ。

剣術習熟：
武器がフライパン、もしくは緋緋色金の剣の場合に弱中強カウンターのダメージ（TFLAG:58のほう）に戦闘素質の25%を上乗せ。
下段狙いの速度低下量に戦闘素質の15%を上乗せ。

体術
美鈴、レミリア、萃香、リグル、文、衣玖、勇儀、華扇、白蓮、こころ

剣術
妖夢、椛、依姫、天子（首輪が武器枠じゃないから微妙だけど）、明羅

……鬼は「技」か？と言われるとうーん。
戦闘素質が高いキャラばっかり持ってるのは意図的。戦闘素質が高いキャラでも非消耗武器を活かせるようにするため。
ぶっちゃけると戦闘素質的に剣士キャラが剣士キャラになれないことへの救済……
だったがキック的にリグルは外せず、設定的に文も外せず。
あんまり補正の掛からないリグルはともかく、文がますます強くなっちゃった。

実際の変化としてはHARDスキュラに力持ちの腕輪と緋緋色金の剣を持たせた天子で実験した所、
ダメージは溜め攻撃＞緋緋色金強攻撃×２＞強弾幕×２。緋緋色金強攻撃には力持ちの腕輪による補正は関係ない。
高戦闘素質キャラに非消耗武器持たせるのが手段の一つとして考えられるレベルになった……かな。
ちなみに急所狙いは現状でも強いので放置、
というか怯み率上乗せしてテストプレイしたら天子がずっと私のターン！やり始めた。
