■触手宮1018用　スライムがうんうんを消化してくれるパッチ

女の子に透明なビューティ的な何かを排泄させたいためだけに作りました。
排泄時に広がって尻穴の中が透けて見えるのっていいよね。


-----

便意あり＋粘装で、ターン終了時にA攻撃されるとスライムさんが汚物を消化してくれます。
汚物が消化されると便意が0になります。

汚物が一定以上(便意3以上)溜まってると、消化したスライムが腸内で急激に増えて尻からスライムひりだし＆ダメージ追加。
(汚物を消化しただけスライムが増殖するんじゃないかなと想定)
ダメージ追加処理は、UNK.ERBの共通ダメージ計算をそのまま拝借しました。

おもらし経験が加算されますが、
ひり出すのは半透明なビューティー的な何かですので汚くありません。下着も脱ぎません。
