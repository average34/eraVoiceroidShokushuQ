東方触手宮1.03.8用 装備変更処理パッチ
※eraT0006655.zipより改変

装備を吟味している時に、ログがものすごい勢いで流れて行くのを防ぐために作りました。
装備を変更するまで、または装備変更画面から戻るまでログが流れないようにしました。

装備解除時・装備変更時はPTメンバの装備一覧のみがログに残ります。
同じスロットに同じものを装備した場合は、消耗武器かつ残弾補充時のみログに残ります。

例外として、マジカルステッキの着脱時については、
装備した時、または装備解除する際に魔法少女の服を着ていた場合はログ削除はされません。


持っていないアイテムの欄（灰色）を選択できない様にもしています。
