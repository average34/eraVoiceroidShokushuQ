これは触手宮1019に、新しい状態異常として「侵食」を追加するパッチです。
イメージとしてはスレ168の92あたりを参考に、身体に寄生された状態となります。

１．発生条件
寄生攻撃、Ａ寄生攻撃を受けた際に一定確率で発生する。
寄生攻撃の場合は従来の寄生の方が優先されるので、すでに寄生されていると可能性が上がる。
Ａ寄生も同じ処理にしようかと思ったが既にＡ寄生でも上書きしてＡ寄生になるのが仕様かどうか分からないので触れず。

２．侵食中の処理
ターン終了時に毎回侵食度が追加されていく。最大値は50。ただし部屋が清浄だと発生しない。
その後、一定の確率で侵食度ごとに各種のイベントが発生する。
・侵食度５未満
侵食度が更に＋１
・侵食度１０未満
侵食度が更に＋１、EPダウン＆ABCVいずれかの経験値増
・侵食度１５未満
侵食度が更に＋１、EPダウン＆ABCVいずれかの経験値増、ABCVMインプラント
・侵食度２０未満
侵食度が更に＋１、EPダウン＆ABCVいずれかの経験値増、全種インプラント、強制EP1
・侵食度２０以上
全種インプラント、強制EP1

３．インプラント
侵食が進むと身体の各所に植え付けるイベントが発生するようになる。
この状態だと各種感覚や中毒が上がりやすくなる上に、侵食イベントでペナルティが重いイベントが発生するようになる。
一例としてはいきなり寄生やＡ寄生されたり、催眠状態になったり、発情したりする。

ただ、射精中毒を上げるインプラントに関してはネタが思いつかず搭載できず。誰かいい案下さい。

４．治療方法
薬酒で侵食度－６、薬草で侵食度－１０（治癒持ちがいれば更に－５）、寄生封じの札で－８、侵食度５以上なら結界で侵食度２分の１。
侵食度が０になればイベントは発生しなくなりインプラントの影響も消えるが、
インプラントされたフラグそのものは消えないためまた侵食されると即座にインプラントが機能する。

５．雑感
毎ターン蓄積のためバランスを考え回復手段多め・発生率低めにしてあるものの、低めにしすぎてショボくなっていないか微妙な所。
従来の寄生攻撃の処理に追加するのではなく新しく寄生攻撃を作ったほうがいいのかもしれない。
