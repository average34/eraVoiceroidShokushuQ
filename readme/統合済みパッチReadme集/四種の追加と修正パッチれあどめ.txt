触手宮1012用四種の追加と修正パッチ （製作者　喚く狂人）Ver1.03

・概要
	「マップのランダム化」「トラップ設置要素の追加」「新アイテムの追加」「キャラの迷い込みのリファクタ」の詰め合わせパッチです
	１．マップのランダム化
		旧マップは「基本固定+一部ランダム」でしたが、このパッチ導入により自動生成になります。アルゴリズムには棒倒し法を用いています。
		基本的な棒倒し法で通路を削除した後、いくつか復旧させるようにしているので、行き止まりだらけということにはならない……とおもいます。多分。
	２．トラップ設置要素の追加
		浄化力を用いてトラップを設置できるようになります。設置や撤去にはターンを消費します。
		戦闘素質の低いキャラなら、これを駆使することで生存率がグッと上がるはずです。逆に、戦闘素質の高いキャラはこんなことする前にとっととブッ飛ばした方が速いです。
	３．新アイテム「すけーるゴーグル」追加
		アイテムとして「すけーるゴーグル」を追加します。使用することで、迷宮内の味方or敵の位置を把握できるスグレモノです。
		使用シーンは非常に多岐にわたる強力なアイテムです。ただし一つまでしか所持できません。
	４．新キャラ迷い込みのリビルド
		「戦意喪失キャラが出た瞬間、すごい勢いでキャラが迷い込み始める」という重篤なバグがあったのと、死に分岐やら何やらあったので勝手に修正しました。
		伴って、コンフィグ対応等しています。FLAGを幾つか使用します。
