ぱにめーしょんのハートマークを使おう！

使い方

基本的に
	CALL HEARTC(0xXXXXXX,*,@@@@@)
	CALL HEARTDB(*)
	CALL HEARTCBW(0xXXXXXX,*,@@@@@)
	CALL HEARTCBW(0xXXXXXX,*,@@@@@,"）")
	CALL HEARTCBW(0xXXXXXX,*,@@@@@,,1)
などで呼び出す

引数0or1の*の部分に数字を入れるとその数だけハートを連打できる。なにも入れなくても一個表示される

@@@@@の部分にはフォント名を入れられる。入れれば対応したフォントになるが基本的に省略でいい

HEARTの後ろの
	Cはカラー。引数0のXXXXXX部分にカラーコードを入れれば好きな色に変えられる。Cを省略すると黒に、コードを省略すると0xC07070（明るい赤）になる
	Dはデフォルトカラー。地の文の色になる。当然Cとは一緒に使えない
	Bは塗りつぶし。省略すると穴あきのハートになる
	Wは改行。
CALL HEARTCBW(,2)だと		赤いハートが二つ並んで」で改行
CALL HEARTCBW(,2,,,1)だと	赤いハートがふた並んでそのまま改行。引数4が0以外だとハートだけ表示して改行する
CALL HEARTCBW(,2,,"）")	だと	赤いハートが二つ並んだあと）で閉じて改行する。"』"を入れれば』にも出来る

主に使うのは
	CALL HEARTCB()
	CALL HEARTCB(,*)
	CALL HEARTCBW(,*)
	CALL HEARTCBW(,*,,"）")
	CALL HEARTCBW(,*,,,1)
