    東方触手宮 手書き地図ツールチップ表示パッチ
    東方触手宮1.05.2修正で動作確認済
                                             作:ふぅりすと
------------------------------------------------------------

■ パッチ内容
手書き地図のメモや、自動書き込みパッチの内容を
表示中のマップにツールチップで表示するパッチです
マウスオーバーで情報の確認が出来ます

表示のちらつきが気になる場合は@PRINT_MAP_SET_TOOLTIP内のTOOLTIP_SETDELAYを良い感じに調整してください

ツールチップの表示をコンフィグでON/OFFしたい等の場合は
@PRINT_MAP_INFO内の呼び出し関数を弄るだけでOKです

東方触手宮 ツールチップ表示修正パッチ

メモや取得情報の中に、マップ表示用文字と同一文字が含まれている場合に発生
<button>と<nonbutton>が入れ子になってしまうのが原因です

一時的にツールチップに表示する文字列を無意味な文字にエンコード
実際に表示する前にデコードする事で対処しました

まず入力されないであろう文字をコード中に記述したため
ファイルのエンコードをSHIFT-JISからUTF-8に変更しました
ご了承ください



このドキュメントは本体同梱不要です

■ 変更詳細
PRINT_MAP_INFO.ERB
	ツールチップタグ埋め込み用の関数と、呼び出しメッセージを作成する関数を新設
	既存コードからの変更点は、@PRINT_MAP_INFOでツールチップ用の関数を呼び出している点のみ
