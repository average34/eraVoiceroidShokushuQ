チュートリアルは実装しましたが、簡単にゲームの解説


このゲームは迷宮の中をうろつきながら、触手モンスターに襲われる様を楽しむゲームです

基本的にはアイテムを探して準備を整えつつ、ボスを探して倒すという流れになります

……が、コンセプト的には全滅してナンボなので、適当に負けてみるのが正しい楽しみ方です


普通に敵を倒していきたい場合、
[100] - 周囲を探索
を実行することで、装備品などを見つけることができます

場所によってはランダムイベントなども起きるので、
移動と探索を繰り返して準備を整えましょう

弱いキャラ（戦闘素質の低いキャラ）は無理に敵を倒そうとせず、ある程度HPを削ったら
さっさと逃げるのも手です。
倒したいのなら強いアイテムや仲間を増やす（３人PTまで可能）方が早いです。



暫く動いていると眠気が溜まってきます
眠気が7以上（要睡眠）になると能力が下がるので、余裕があるうちに寝ておきましょう

眠気が15以上になると気絶してしまい、睡姦されたりするので
そんな事は起きないように気をつけましょう



敵を倒しても基本的に経験値は溜まりませんが、
様々な行動をする度に各能力の経験値が溜まっていきます。（基本的に失敗した方が溜まりやすいです）
最初は経験値表示をONにして、何をすると何が上がるのか調べてみると良いでしょう

慣れてくると邪魔なだけなので、コンフィグで表示をOFFにするのがオススメです




最後に能力値の説明だけ。
HP … 高いと戦闘で有利になります。0になっても死んだりしませんが、様々な確率にペナルティを受けます
MP … いわゆる真のHPです。心にダメージを負うと減っていき、0になると戦闘不能になります
EP … 快感に耐えられる度合です。0になると絶頂し、MPが減ったりします
CP … 服の耐久度です。0になると裸になり、恥ずかしさから戦闘中MPが徐々に減ったり、EPが減りやすくなったりします
TP … テンションの高さです。100が基準値で、0～200の間で変動します
      高いほど攻撃力が上がったり、命中率が高くなったりします
      低いほど防御力が高くなったり、命中率が凄く高くなったり、逃げやすくなったりします
      中途半端な状態が一番使えませんが、時間経過や睡眠で100(性格などで多少前後)に近づきます

パーティー全員のMPが0になると全滅します
そのまま敵に生命力（最大HP）を吸われていき、0になるとキャラロストです（コンフィグで設定可能）

吸われている間でも、通りがかったNPCに助けを求めたりすることは可能です
現状ではパーティー枠が埋まっていると誰にも助けてもらえないので注意してください
